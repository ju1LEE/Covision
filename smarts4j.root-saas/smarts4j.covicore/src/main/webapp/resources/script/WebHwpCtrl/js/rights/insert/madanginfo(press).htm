필드 입력

필드란 사용자 입력을 허용하는 일종의 양식 개체로 문서에 필드 입력란을 추가하여 다른 사용자가 간단한 내용을 직접 입력하게 할 수 있습니다.

따라 하기

  1. [입력-필드 입력]을 실행합니다.
  2. [입력할 내용의 안내문]에 "이곳을 마우스로 누르고 내용을 입력하세요."라고 처음 값이 입력되어 있습니다.
  3. [넣기] 단추를 누르면 편집 화면에 낫표()로 묶인 "이곳을 마우스로 누르고 내용을 입력하세요." 메시지가 나타납니다.
  4. 누름틀 안내문의 내용을 고치려면 누름틀 안에 커서를 놓고 마우스 오른쪽 단추를 누른 다음 [누름틀 고치기]를 누릅니다.

자세한 정보

[필드 입력] 대화 상자의 [누름틀] 탭

입력할 내용의 안내문

안내문 입력 상자에 입력한 내용은 편집 화면에 표시됩니다.

편집 화면에 나타난 안내문을 마우스로 누르면, 안내문이 사라지고 낫표()로 묶인 누름틀 영역이 표시되며 글자를 입력할 수 있는 상태가 됩니다.

필드 이름

삽입한 누름틀 필드의 이름을 입력합니다. 필드 이름을 지정하면 여러 개의 누름틀이 있을 때 각 누름틀을 구분하기 쉽습니다.

양식 모드에서 편집 가능

양식 모드에서도 입력된 글상자를 편집할 수 있도록 허용합니다. 양식 모드란 다른 사람이 문서 내용을 변경할 수 없도록 문서를 보호하며 양식 모드에서 편집 속성을 가진 개체만 편집이 가능하도록 허용하는 모드입니다. 이 기능은 외부 프로그램에서 웹한글을 제어할 때 사용합니다.

참고

누름틀 고치기

본문에 입력한 누름틀의 내용을 고치려면, 누름틀 바로 앞에 커서를 놓거나 누름틀 안에 커서를 놓고 마우스 오른쪽 단추를 누른 다음 [누름틀 고치기]를 실행합니다.

누름틀 내용만 블록으로 설정하기

누름틀 바로 앞이나 누름틀 안에 커서를 놓고 <F11>을 누르면 누름틀 안의 내용만 한 번에 블록으로 설정해 줍니다. 이 상태에서 다시 한 번 <F11>을 누르면 커서 위치보다 앞으로 거슬러 올라가면서 "누름틀, 그리기 개체, 그림, 표, 수식, 글맵시" 중에 가장 먼저 나오는 개체를 선택해 줍니다.

현재 선택된 개체나 누름틀이 문서의 맨 처음에 있는 개체인 경우에는 <F11>을 누르면 개체나 누름틀 선택이 해제됩니다.
