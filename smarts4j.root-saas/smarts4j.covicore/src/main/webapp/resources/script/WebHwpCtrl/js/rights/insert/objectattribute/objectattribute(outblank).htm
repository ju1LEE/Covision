개체 속성: 여백/캡션

개체를 삽입한 다음 <P>를 누르거나 [서식-개체 속성]을 실행하면 나타나는 [개체 속성] 대화 상자의 [여백/캡션] 탭에서는 개체 가장자리의 바깥쪽부터 본문까지의 간격을 정하거나 캡션의 위치, 크기, 개체와의 간격 등을 설정합니다.

따라 하기

캡션 넣기와 캡션 위치 조정하기

  1. 캡션 넣기를 수행할 그림을 선택한 상태에서 마우스 오른쪽 단추를 눌러 빠른 메뉴가 나타나면 [캡션 넣기]를 선택합니다.
  2. 그림의 아래쪽에 "그림 1"이 자동으로 나타나면 그림의 제목을 입력합니다.
  3. 그림을 선택한 상태에서 <P>를 누른 다음 [개체 속성] 대화 상자가 나타나면 [여백/캡션] 탭을 선택합니다.
  4. [캡션] 항목에서 '오른쪽 아래'를 선택하고 [설정] 단추를 누릅니다.
  5. "그림 1"이라고 표시된 캡션의 위치가 그림의 아래쪽에서 그림의 오른쪽으로 바뀝니다.

자세한 정보

[개체 속성] 대화 상자의 [여백/캡션] 탭

바깥 여백
  • 왼쪽

  • 개체 가장자리의 왼쪽과 본문 사이의 간격을 조정합니다.

  • 오른쪽

  • 개체 가장자리의 오른쪽과 본문 사이의 간격을 조정합니다.

  • 위쪽

  • 개체 가장자리의 위쪽과 본문 사이의 간격을 조정합니다.

  • 아래쪽

  • 개체 가장자리의 아래쪽과 본문 사이의 간격을 조정합니다.

캡션
  • 위치

  • 캡션 내용이 들어갈 위치는 개체의 테두리 바깥쪽입니다.

    [위, 아래, 왼쪽 위/가운데/아래, 오른쪽 위/가운데/아래, 없음] 중에서 캡션의 위치를 선택할 수 있습니다.

  • 크기

  • 캡션의 위치가 [위]나 [아래]일 때에는 캡션의 크기는 개체의 가로 크기와 같습니다. 하지만 캡션의 위치를 개체의 [왼쪽]이나 [오른쪽]으로 설정했을 때에는 캡션의 크기를 지정할 수 있습니다.

  • 개체와의 간격

  • 개체와 캡션 내용 사이의 간격을 얼마큼 띄울 것인지 정합니다. 처음 값은 3mm입니다. 한/글에서는 개체와의 간격에 음수 값을 지원하므로, 필요한 경우 [개체와의 간격]에 음수 값을 입력하여 사용할 수 있습니다.

  • 여백 부분까지 너비 확대

  • 개체에 바깥 여백을 설정하더라도 캡션은 바깥 여백 부분을 제외하고 순수한 개체 영역부터 시작됩니다. 이 항목을 설정해 놓으면 왼쪽이나 오른쪽 바깥 여백부터 캡션을 시작합니다.

  • 한 줄로 입력

  • 이 항목을 선택하면 캡션 영역에 글을 입력하면서 커서가 오른쪽 경계선에 닿아도 커서가 다음 줄로 넘어가지 않고 한 줄에 모두 입력됩니다.

참고

바깥 여백과 캡션

개체에 바깥 여백을 설정하더라도 캡션은 바깥 여백 부분을 제외하고 순수한 개체 영역부터 시작됩니다. [캡션-여백 부분까지 너비 확대]를 켜 놓으면, 왼쪽이나 오른쪽 바깥 여백부터 캡션을 시작합니다.

개체와의 관계

바깥 여백은 개체와 본문 글자 사이의 간격을 정하는 것입니다. 따라서 [글자처럼 취급]하지 않은 개체와 개체 사이에는 아무런 영향을 주지 않습니다.

크기

개체에 여백값을 주더라도 개체의 크기는 변하지 않습니다.
