문서 끼워 넣기 <Ctrl+O>

편집하고 있는 문서의 커서 위치에 선택한 문서 파일을 삽입합니다.

이 기능을 실행하여 서로 다른 파일로 만들어진 두 개 이상의 문서를 하나로 합칠 수 있습니다.

따라 하기

  1. 원본 문서 '문서1'을 편집하는 도중에 [입력-문서 끼워 넣기]를 실행합니다.
  2. [문서 끼워 넣기] 대화 상자가 나타나면 현재 문서에 끼워 넣으려는 대상 문서 '문서2'를 지정합니다.
  3. [글자 모양 유지], [문단 모양 유지], [쪽 모양 유지]를 선택하고 [넣기] 단추를 누릅니다.
  4. 커서가 있던 자리에 '문서2'의 파일 내용 전체가 글자, 문단, 쪽 모양을 그대로 유지한 상태로 삽입됩니다.

자세한 정보

[문서 끼워 넣기] 대화 상자

[파일 선택] 단추

현재 편집 중인 문서에 삽입할 다른 문서를 선택합니다.

쪽 모양 유지

현재 커서 위치를 기준으로 앞뒤로 구역을 나눈 다음 나뉜 구역 사이에 끼워 넣는 파일의 쪽 모양을 그대로 유지하면서 파일을 끼워 넣습니다.

글자 모양 유지

끼워 넣을 문서의 글자 모양을 그대로 유지한 상태로 끼워 넣습니다.

원본 문서에 대상 문서와 동일한 스타일 이름이 있더라도 대상 문서의 글자 모양을 유지하여 끼워 넣기를 수행합니다.

문단 모양 유지

끼워 넣을 문서의 문단 모양을 그대로 유지한 상태로 끼워 넣습니다.

원본 문서에 대상 문서와 동일한 스타일 이름이 있더라도 대상 문서의 문단 모양을 유지하여 끼워 넣기를 수행합니다.

스타일 유지

끼워 넣을 문서의 스타일 이름 뒤에 '사본(숫자)'를 넣은 상태로 끼워 넣습니다.

원본 문서에 대상 문서와 동일한 스타일 이름이 있을 경우 대상 문서의 스타일 이름 뒤에 사본(숫자)를 넣어 끼워 넣기를 수행합니다.

참고

파일 열기와 끼워 넣기의 차이점

[파일 열기]는 새 문서 창(문서 탭)을 열고 문서를 불러와 편집하는 것이고, [문서 끼워 넣기]는 선택한 문서를 불러서 현재 문서의 커서 위치에 그 내용을 합치는 것입니다.

파일 이름

[문서 끼워 넣기]를 해도 현재 문서의 파일 이름은 바뀌지 않고 그대로 유지됩니다.

끼워 넣는 파일의 스타일

[문서 끼워 넣기]를 하면 그 문서의 본문 내용은 물론 편집된 모든 내용이 그대로 현재 문서에 포함됩니다.

블록 설정하고 문서 끼워 넣기

현재 문서 내용의 일부를 블록으로 설정한 상태에서 [문서 끼워 넣기]를 실행하면, 블록으로 설정한 부분을 지우고 끼워 넣는 문서의 내용으로 덮어 씁니다.

이것은 블록을 설정한 상태에서 새로운 내용을 입력할 때, 블록 부분을 지우고 새 내용이 덮어 써지는 것과 같습니다.
