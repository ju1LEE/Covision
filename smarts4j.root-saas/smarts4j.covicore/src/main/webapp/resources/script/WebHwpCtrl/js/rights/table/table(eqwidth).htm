셀 너비를 같게 <W>

[셀 너비를 같게]는 표에서 너비가 서로 다른 셀들의 너비를 모두 같게 만들어 주는 기능입니다.

따라 하기

  1. 셀의 너비를 같게 만들려는 셀들을 셀 블록으로 설정합니다.
  2. [표-셀 너비를 같게]를 실행합니다.
  3. 셀 블록으로 설정한 셀들의 너비가 같아집니다.

주의

제한

[셀 너비를 같게]를 할 때, 선택된 셀 블록의 형태는 반드시 직사각형이어야 합니다. 만약 불연속적인 셀 블록을 설정했거나 직사각형 모양이 아닌 셀 블록을 설정했을 때에는 [셀 너비를 같게]를 할 수 없습니다.
