줄/칸 지우기

표를 편집하다 보면 줄 수와 칸 수가 처음에 지정한 것보다 남는 경우가 생깁니다. 따라서 표를 만드는 과정에서는 필요 없는 부분의 줄이나 칸을 지우는 일을 해야 합니다.

따라 하기

  1. 지우고자 하는 줄이나 칸 중의 어느 한 셀에 커서를 놓습니다.
  2. [표-줄/칸 지우기]를 누른 다음 [줄 지우기] 또는 [칸 지우기] 하나를 선택합니다.
  3. 현재 커서가 있는 셀이 포함된 줄 또는 칸 전체가 삭제됩니다.
  4. 지우려는 줄 또는 칸의 개수만큼 위 과정을 반복합니다.

참고

셀 블록

셀 블록을 설정해 [줄/칸 지우기]를 수행할 때에는 반드시 하나의 셀만 셀 블록으로 설정해야 합니다. 2칸 이상의 셀을 셀 블록으로 설정하면 [줄/칸 지우기]를 수행할 수 없습니다.

지우기와 되돌리기

지금 [줄/칸 지우기]를 실행한 것이 실수였다면, 바로 [되돌리기]를 실행하십시오. 바로 전에 실행한 [줄/칸 지우기]의 동작을 취소시켜주므로 지운 내용이 되살아납니다.
