배율

편집하는 문서의 실제 크기를 바꾸지 않고 화면 배율을 변경할 수 있습니다.

따라 하기

  1. [보기-확대] 또는 [보기-축소]를 실행합니다.
  2. 현재 문서의 한 쪽 분량이 한 화면으로 볼 수 있는 배율로 축소되거나 확대되어 나타납니다.
  3. 다시 [보기-배율]을 실행해 [100%]를 선택하면 화면이 100% 배율로 조절되어 보입니다.

자세한 정보

배율

[50%, 75%, 100%, 125%, 150%, 200%, 300%, 폭 맞춤, 쪽 맞춤] 중에서 배율을 선택하거나, 사용자가 원하는 화면 배율을 지정할 수 있습니다.

  • 폭 맞춤
    편집 용지의 크기와 관계없이 현재 용지의 너비가 문서 창의 너비에 맞도록 축소하거나 확대합니다.
  • 쪽 맞춤
    편집 용지의 크기와 관계없이 현재 용지 한 쪽 분량을 한 화면에 모두 볼 수 있는 배율로 축소하거나 확대합니다.
  • 50% - 300%
    편집 화면을 50% - 300%로 확대 또는 축소합니다. 100%는 편집 화면을 확대하거나 축소하지 않은 실제 크기입니다

참고

상황 선 [확대/축소] 이용하기

[보기-확대/축소]를 실행하지 않고도 [상황 선-확대/축소] 아이콘()을 이용하여 확대 배율을 쉽게 바꿀 수 있습니다.

단축키 사용

배율 단축키
쪽 맞춤 Ctrl+G,P
폭 맞춤 Ctrl+G,W
화면 100% Ctrl+G,Q
단계 확대 Shift+(Num)+
단계 축소 Shift+(Num)-

문서 편집과는 무관

[확대/축소]는 단지 화면 배율만을 조절하는 것이므로, 어떤 화면 배율을 선택하더라도 문서 편집 기능에는 제약이 따르지 않습니다.
