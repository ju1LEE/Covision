셀 나누기 <S>

현재 커서가 있는 셀 또는 셀 블록으로 설정된 셀들을 여러 개의 셀로 나눕니다.

따라 하기

  1. 여러 개로 나눌 셀에 커서를 놓고 [표-셀 나누기]를 실행합니다.
  2. [셀 나누기] 대화 상자가 나타나면 셀을 몇 개의 칸과 줄로 나눌 것인지 2~63 사이에서 지정합니다.
  3. [나누기] 단추를 누릅니다.
  4. 현재 셀을 지정한 숫자만큼 나눕니다. 이때 나누어지는 셀의 속성은 원래의 셀 속성을 따라갑니다.

자세한 정보

[셀 나누기] 대화 상자

줄/칸 나누기

  • 줄 개수

  • 셀을 몇 개의 줄로 나눌지 줄 수를 입력하거나 증감 단추를 눌러 줄 수를 지정합니다.

  • 칸 개수

  • 셀을 몇 개의 칸으로 나눌지 칸 수를 입력하거나 증감 단추를 눌러 칸 수를 지정합니다.

선택 사항

  • 줄 높이를 같게 나누기

  • 줄을 나눌 때 나뉜 줄의 높이가 모두 같게 하려면 [줄 높이를 같게 나누기]를 선택하고 줄을 나눕니다.

  • 셀을 합친 후 나누기

  • 여러 개의 셀을 셀 블록으로 설정해 [셀 나누기]를 할 때에는 각각의 셀을 나눌 것인지, 셀 블록으로 설정한 셀들을 하나로 합친 후 나눌 것인지 선택할 수 있습니다. 이 항목을 선택하고 셀을 나누면, 셀들을 먼저 하나로 합친 다음 정해진 숫자만큼 줄/칸을 나눕니다.

참고

줄/칸 동시 나누기

셀을 나눌 때 "줄"과 "칸"을 모두 선택하면 줄과 칸을 동시에 나눌 수 있습니다. 하지만 "줄"과 "칸"을 모두 선택하지 않을 수는 없습니다. 즉, "줄"이나 "칸" 중에서 하나만 나누던지 둘 다 나누어야 합니다.

주의

제한

[셀 나누기]를 할 때, 선택된 셀 블록의 형태는 반드시 직사각형이어야 합니다. 만약 불연속적인 셀 블록을 설정했거나 직사각형 모양이 아닌 셀 블록을 설정했을 때에는 [셀 나누기]를 실행할 수 없습니다.
