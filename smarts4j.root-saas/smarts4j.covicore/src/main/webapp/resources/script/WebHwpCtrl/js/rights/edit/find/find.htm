찾기 <Ctrl+F>

현재 편집하고 있는 문서에서 특정한 낱말(문자열)을 찾습니다.

따라 하기

  1. [편집-찾기-찾기]를 실행합니다.
  2. [찾기] 대화 상자가 나타나면 [찾을 내용]에 현재 문서의 내용 중에 찾고자 하는 문자열을 입력합니다.
  3. 마우스로 [다음 찾기] 단추를 누르거나, 글자판의 <Enter>를 누릅니다.
  4. [찾을 내용]에 해당하는 낱말을 찾아 블록으로 감싸 보여 줍니다. 만일 찾고자 하는 문자열이 없다면, 찾는 내용이 없다고 알려 줍니다.
  5. 마우스로 [다음 찾기] 단추를 누르거나 글자판의 <Enter>를 누르면 같은 내용을 다음에서 계속 찾습니다.

자세한 정보

[찾기] 대화 상자

찾을 내용

찾을 내용을 입력합니다.

찾을 방향

문서에서 입력한 내용을 찾을 때 찾을 방향을 지정합니다. 원하는 내용을 보다 빨리 찾을 수 있습니다.

[다음 찾기] 단추

[다음 찾기] 단추를 누를 때마다 문서 내용 중에서 [찾을 내용]과 일치하는 낱말을 순서대로 찾아서 블록으로 감싸 보여 줍니다.

[닫기] 단추

찾기 실행을 종료하고 [찾기] 대화 상자를 닫습니다.

[찾기]는 주로 여러 번 반복하여 실행하는 경우가 많기 때문에 사용자가 [닫기] 단추를 눌러 대화 상자를 닫을 때까지는 자동으로 닫히지 않고 화면에 계속 남아 있습니다.

참고

블록 설정하고 찾기

문서의 일부에서만 찾기를 하려면, 먼저 원하는 부분을 블록으로 설정한 다음 [찾기]를 실행합니다.
블록으로 설정된 부분에서 찾기를 끝낸 뒤에는 나머지 내용에서도 찾기를 계속할 것인지 사용자에게 확인합니다.

실행 범위

현재 커서의 위치에서 [찾을 방향]에 따라 [아래쪽]이면 문서의 끝까지, [위쪽]이면 문서의 처음까지 찾고, 나머지 부분에서도 계속 찾을지 사용자에게 확인합니다.
[문서 전체]이면 현재 커서의 위치부터 문서의 끝까지 찾은 다음 이어서 문서의 처음부터 현재 커서 위치까지 찾습니다.
