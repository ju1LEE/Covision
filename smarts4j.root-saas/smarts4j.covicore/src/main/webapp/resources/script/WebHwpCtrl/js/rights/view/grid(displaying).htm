격자 보기

'격자(grid)'는 편집 화면에 가로 세로 일정한 간격으로 점을 찍거나 선을 그려 주는 것입니다. 격자를 이용하면 그림을 그리거나 그려진 그림을 편집할 때 정확히 간격을 맞추어 세밀한 편집을 할 수 있습니다.

따라 하기

격자 보이기/숨기기

  1. [보기-격자 보기]를 실행합니다.
  2. [격자 보기] 메뉴 앞에 선택 표시()가 나타나고 문서 창에 격자 선이 표시됩니다.

  3. [보기-격자 보기]를 다시 실행합니다.
  4. [격자 보기] 메뉴 앞에 선택 표시가 사라지고 문서 창에서도 격자가 사라집니다.

참고

격자 인쇄

[격자 보기]를 선택해 놓더라도 편집 화면에서 그림을 그리거나 그려진 그림을 편집할 때에만 도움을 줄 뿐, 문서를 인쇄하면 격자는 인쇄되지 않습니다.
