복사하기 <Ctrl+C>

블록으로 선택한 부분을 클립보드에 저장합니다. 저장된 내용은 [편집-붙이기]를 실행하여 원하는 위치에 붙여넣을 수 있습니다. 문서의 특정 내용을 다른 곳에서 동일하게 사용할 때 주로 사용합니다.

따라 하기

  1. 문서에서 복사하려는 내용을 블록으로 설택합니다.
  2. [편집-복사하기] 또는 <Ctrl+C>를 누릅니다. 선택한 내용이 클립보드에 저장됩니다.

참고

단축키 사용

일부 웹 환경에서 클립보드에 접근하려면 반드시 단축키를 사용해야 합니다. 따라서 오려 두기, 복사하기, 붙이기 기능을 실행하려면 다음 단축키를 사용하시기 바랍니다. Internet Explorer에서는 단축키 대신 [편집-복사하기]를 사용할 수 있습니다.

  • 오려 두기: Ctrl + X
  • 복사하기: Ctrl + C
  • 붙이기: Ctrl + V

복사하기와 오려 두기

  • 복사하기: 블록으로 설정된 부분을 클립보드에 저장하는 작업만 수행합니다.
  • 오려 두기: 블록으로 설정된 부분을 클립보드에 저장하면서 현재 위치에서는 그 내용을 지웁니다.

클립보드

클립보드(clipboard)는 윈도우에서 응용 프로그램 사이의 데이터 전송을 위해 사용되는 임시 기억 장소입니다.

일반적으로 응용 프로그램에서 [복사하기]나 [오려 두기]를 실행하여 데이터를 클립보드에 임시로 저장하고, [붙이기]를 실행하여 클립보드에 임시 저장된 데이터를 문서에 삽입합니다.

표 안에서의 복사하기

셀 블록을 설정하고 [복사하기]를 하면 표 안의 내용뿐만 아니라 블록으로 설정한 표의 틀까지 복사됩니다.

표 안에서 [붙이기]를 하면 [셀 붙이기] 대화 상자가 나타납니다. 여기서 복사한 표를 끼워 넣을 것인지, 덮어 쓸 것인지, 셀 안에 표로 넣을 것인지 붙일 방법을 선택합니다.

활용

대화 상자의 입력 상자에서도 사용

[복사하기], [오려 두기], [붙이기]는 대화 상자에서 [파일 이름]과 같이 사용자가 내용을 직접 입력할 수 있는 대부분의 입력 상자에서도 쓸 수 있습니다.
