직사각형

편집 화면에 사각형을 그립니다.

따라 하기

  1. [입력-도형]을 누릅니다.
  2. [직사각형] 아이콘()을 누릅니다.

참고

사각형 모서리

사각형의 모서리는 일반적으로 직각 모양입니다. [서식-개체 속성]을 실행해 [선] 탭에서 [끝 모양]을 둥근 모서리로 변경할 수 있습니다.

개체 속성 변경하기

입력된 개체의 속성을 변경하려면 [서식-개체 속성]을 실행하거나 개체의 테두리 부분을 마우스로 두 번 눌러 [개체 속성] 대화 상자를 불러옵니다. [개체 속성] 대화 상자를 이용하여 개체의 크기와 위치 조절, 개체의 테두리 모양과 채우기 효과 등을 변경할 수 있습니다.

관련 기능

도형, 개체 속성, 캡션 넣기
