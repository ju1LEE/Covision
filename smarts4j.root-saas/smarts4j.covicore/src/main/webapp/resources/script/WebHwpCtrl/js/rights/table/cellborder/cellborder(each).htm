각 셀마다 적용

표의 각 셀을 에워싸는 테두리의 모양을 바꾸거나 대각선 긋기, 또는 셀마다 배경색과 배경 그림 등을 넣어 표를 보기 좋게 꾸밉니다.

참고

셀 단위로 적용

셀 테두리/배경은 각 셀 단위로 적용됩니다. 셀 블록을 설정하고 셀 배경을 지정하더라도 모든 셀에 걸치는 배경이 적용되는 것이 아니라, 각 셀마다 지정한 배경이 적용됩니다.

모든 셀에 걸치는 배경을 만들고 싶다면 [셀 테두리/배경-하나의 셀처럼 적용] 기능을 사용하시기 바랍니다.

표/테두리 배경

표의 모든 셀에 걸치는 배경을 만들고 싶을 때에는 [셀 테두리/배경] 대화 상자에서 [적용 범위-모든 셀] 단추를 누르십시오.

단축키

셀 블록 상태에서 다음과 같은 단축키를 사용하여 셀 테두리/배경 작업을 쉽게 할 수 있습니다.

  • 셀 테두리: <L>
  • 셀 배경: <C>
