단축키 일람

사용자의 편의를 위해 데스크톱 한/글에서 제공하던 일부 단축키를 그대로 지원합니다. 지금까지 데스크톱 한/글의 단축키를 사용해 왔다면 비록 문서 편집 환경이 바뀌었다 하더라도 별다른 어려움 없이 단축키를 사용할 수 있습니다.

'단축키'는 <Ctrl>이나 <Alt> 또는 <Shift>와 함께 해당 영문 글쇠를 눌러 특정 기능을 바로 실행할 때 사용합니다. '겹글쇠 단축키'는 <Ctrl+N,T>와 같이 <Ctrl>과 <N>을 동시에 눌렀다가 손을 뗀 후 다시 <T>를 누르면 기능이 실행되는 단축키를 의미합니다.

프로그램에서 제공하는 단축키가 브라우저의 기본 단축키와 충돌하는 경우 브라우저 단축키가 우선적으로 동작합니다.
