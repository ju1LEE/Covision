문단 테두리

문단을 에워싸는 테두리를 넣어 문단을 보기 좋게 만듭니다.

따라 하기

  1. 문단 테두리나 배경을 지정할 문단에 커서를 놓고, [서식-문단 모양]을 실행합니다.
  2. [문단 모양] 대화 상자가 나타나면 [테두리-종류] 목록 단추를 눌러 두 번째 항목의 실선을 선택하고, [굵기]는 0.5mm로 지정합니다.
  3. [설정] 단추를 누르면 현재 커서가 있는 문단에 테두리가 그려집니다.

참고

테두리 모양

다양한 선 종류의 문단 테두리를 선택할 수 있으며, 각각의 테두리 종류에 대하여 굵기와 색깔을 지정할 수 있습니다. 선 굵기는 두께 목록 중에서 선택만 할 수 있고, 굵기 값을 직접 입력해서 두께를 정할 수는 없습니다.

문단 테두리 연결

두 개 이상의 문단에 대하여 현재 문단과 이어지는 다음 문단들을 하나의 문단 테두리로 연결해 줍니다.

예를 들어, 단, 왼쪽/오른쪽 문단 여백을 5pt, 위/아래 문단 간격을 5pt로 설정한 문단에 [문단 테두리]를 적용하는 경우 [문단 테두리 연결] 항목의 선택 여부에 따라 다음과 같이 다른 결과가 나타납니다.

[문단 테두리 연결]을 선택한 경우

[문단 테두리 연결]의 선택을 해제한 경우

주의

문단 테두리 해제

테두리를 해제하기 위해서는 [테두리-종류]에서 "선 없음"을 선택하고, 테두리를 해제할 방향의 테두리 단추를 누르십시오.
