개체 모양 복사/붙이기 <Ctrl+Alt+C>/<Ctrl+Alt+V>

[개체 모양 복사]와 [개체 모양 붙이기]는 그림 개체 또는 그리기 개체의 선 종류, 채우기 모양, 개체 크기 등 개체의 모양을 복사하여 다른 개체에 간편하게 붙여 넣는 기능입니다. 특정한 개체의 모양을 반복적으로 자주 지정해야 하는 경우 매우 간편하게 활용할 수 있습니다.

따라 하기

개체 모양 복사/붙이기

  1. 편집 화면에서 모양을 복사하려는 개체를 선택합니다.

  2. 마우스 오른쪽 단추를 누른 다음 빠른 메뉴에서 [개체 모양 복사]를 누릅니다.
  3. [개체 모양 복사] 대화 상자가 나타나면 복사할 항목을 선택한 다음 [복사] 단추를 누릅니다.
    여기에서는 [공통 모양 복사-선 모양]과 [그리기 모양 복사-채우기] 항목을 선택합니다.
  4. 복사한 모양을 덮어쓸 다른 개체를 선택합니다.

  5. 마우스 오른쪽 단추를 누른 다음 빠른 메뉴에서 [개체 모양 붙이기]를 누릅니다.

자세한 정보

[개체 모양 복사] 대화 상자

공통 모양 복사

  • 선 종류

    [서식-개체 속성-선]에서 설정한 개체의 선 종류를 복사합니다.

  • 개체 크기

    개체의 너비와 높이를 복사합니다.

그리기 모양 복사

  • 그림자

    그리기 개체의 그림자 모양, 위치, 색을 복사합니다.

  • 채우기

    [서식-개체 속성-채우기]에서 설정한 그리기 개체의 면 색, 면 무늬 또는 채우기 효과를 복사합니다.

그림 모양 복사

  • 그림 효과

    그림 개체의 흑백 효과, 회색조, 워터마크 효과, 그림 반전 효과 등 그림 효과를 복사합니다.

참고

[모양 복사]와 [개체 모양 복사]

  • 모양 복사

    [모양 복사] 기능은 현재 커서 위치의 본문 모양(글자 모양, 문단 모양, 스타일)을 복사하여 다른 본문에 똑같이 적용하거나, 표에서 블록 잡은 셀의 본문 모양뿐만 아니라 셀 모양(셀 속성, 셀 테두리, 셀 배경)까지 복사해 두었다가 다른 셀에 그대로 똑같이 덮어 쓰고자 할 때 사용합니다.

    모양 복사/붙이기 기능의 단축키는 <Ctrl+Alt+C>로 동일합니다.

  • 개체 모양 복사

    [개체 모양 복사] 기능은 그리기 개체 또는 그림 개체의 모양을 복사하여 다른 그리기 개체나 그림 개체에 적용하고자 할 때 사용합니다.

    개체 모양을 복사하여 붙일 때에는 [개체 모양 복사] 기능의 단축키로는 <Ctrl+Alt+C>, [개체 모양 붙이기] 기능의 단축키로는 <Ctrl+Alt+V>를 각각 사용합니다.
