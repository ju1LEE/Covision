다른 이름으로 저장

편집 중인 문서를 다른 이름으로 저장하려면 다음을 실행합니다.

  1. 파일-다른 이름으로 저장을 누릅니다.
  2. 다른 이름으로 저장 대화 상자에서 파일을 저장할 폴더를 선택합니다.
  3. 파일명에 새로운 파일 이름을 입력합니다.
  4. 확인 단추를 누릅니다.

참고

상태 메시지 표시

문서를 저장하는 동안 저장 상태에 대한 메시지는 편집 화면 오른쪽 상단에 표시됩니다.

다른 이름으로 저장 메뉴 지원 안 함

기능 구현 환경에 따라 다른 이름으로 저장 메뉴가 나타나지 않을 수 있습니다. 메뉴에 대한 지원 여부는 구현 환경을 따르며 사용자 임의로 변경할 수 없습니다.
