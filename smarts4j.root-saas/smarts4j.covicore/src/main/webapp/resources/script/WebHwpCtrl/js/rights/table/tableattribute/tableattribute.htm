표/셀 속성 <P>

본문에 삽입한 표의 크기와 위치, 여백, 배경, 테두리 등을 설정할 수 있습니다.

따라 하기

표 속성 설정하기

  1. [표-표 만들기]를 실행하여 줄/칸을 지정하고 [만들기] 단추를 눌러 표를 만듭니다.
  2. 표를 선택한 상태에서 <P>를 누르거나 [표-표/셀 속성]을 실행합니다.
  3. [표/셀 속성] 대화 상자가 나타나면 [배경] 탭을 선택합니다.
  4. [색] 항목에서 [면 색]을 지정한 다음 [설정] 단추를 누릅니다.
  5. 편집 화면의 표에 배경이 적용됩니다.

셀 속성 설정하기

  1. [표-표 만들기]를 실행하여 줄/칸을 지정하고 [만들기] 단추를 눌러 표를 만듭니다.
  2. 첫 번째 셀에 "한글과컴퓨터"라고 입력합니다.
  3. [표-표/셀 속성]을 실행합니다.
  4. [표/셀 속성] 대화 상자가 나타나면 [셀] 탭을 선택합니다.
  5. [안 여백 지정]을 선택하고 위쪽, 아래쪽 여백을 각각 5.00mm 로 지정한 다음 [설정] 단추를 누릅니다.
  6. "한글과컴퓨터"라고 입력한 셀의 위아래 여백이 늘어납니다.
