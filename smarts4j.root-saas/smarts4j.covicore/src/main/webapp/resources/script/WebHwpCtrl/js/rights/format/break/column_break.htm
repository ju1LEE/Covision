단 나누기 <Ctrl+Shift+Enter>

단의 내용이 다 차지 않았더라도 다음 단으로 커서를 옮기고자 할 때에 [단 나누기]를 이용하여 커서 이후의 내용을 다음 단으로 분리시킬 수 있습니다.

따라 하기

  1. 새로운 단으로 시작하고 싶은 곳에 커서를 놓고 [쪽-단 나누기]를 실행합니다.
  2. 단이 나뉘면서 다음 단의 맨 앞에 커서가 위치합니다.

참고

단 나누기 취소

단 나누기를 취소하려면 [쪽-단 나누기]를 실행한 자리 앞이나 뒤에서 <Delete>나 <Backspace>를 누릅니다. 이때 '강제 단 나누기' 표시만 없어지고 앞뒤의 문단을 하나로 합치지는 않습니다.

1단에서 단 나누기

다단 편집을 하지 않은 1단의 문서에서 [단 나누기]는 강제로 쪽 나누기를 한 것과 같습니다.
