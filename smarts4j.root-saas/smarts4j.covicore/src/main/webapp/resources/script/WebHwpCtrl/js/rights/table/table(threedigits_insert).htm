자릿점 넣기

표 안의 숫자에 일괄적으로 1,000단위마다 자릿점을 넣을 때 사용합니다.

[1,000 단위 구분 쉼표-자릿점 넣기]를 하면 숫자를 감지하여 셀 블록으로 지정한 영역의 1,000단위가 넘는 모든 숫자에 1,000단위마다 자릿점(,)을 넣어 줍니다. 이렇게 삽입한 1,000단위 자릿점(,)은 [자릿점 빼기]로 일괄 삭제할 수 있습니다.

따라 하기

  1. [표-표 만들기]를 실행합니다.
  2. [표 만들기] 대화 상자가 나타나면 줄 수와 칸 수를 지정하고 [만들기] 단추를 누릅니다.
  3. 만들어진 표에 1,000단위 숫자를 입력합니다.
  4. 표 안에 커서를 넣고 <F5>를 세 번 눌러 표 전체에 [셀 블록]을 설정합니다.
  5. [표-1,000 단위 구분 쉼표-자릿점 넣기]를 누릅니다.
  6. 셀 블록으로 설정된 내용 중 숫자가 입력된 셀에 1,000단위마다 자릿점(,)이 입력됩니다.

참고

영역 설정

1,000단위마다 자릿점을 입력할 때 영역 설정은 <F5>를 눌러 [셀 블록]을 설정했을 때 동작합니다. 표밖에 커서를 두고 <F3> 또는 <F4>를 눌러 블록 또는 구역을 설정한 경우엔 [자릿점 넣기] 기능이 올바르게 동작하지 않습니다.

자릿점이 입력되지 않는 경우

  1. 세 자리 이하 연속되는 숫자
    예) 200
  2. 숫자와 마침표(소수점)와 숫자가 연속한 경우 마침표 뒤의 숫자 부분 (소수부)
    예) 200.55555
  3. 날짜/시간 코드를 입력했을 경우에 연도 네 자리
    예) 2013년 10월 9일
  4. 날짜/시간 코드의 필드 사이에 들어간 네 자리 이상 연속된 숫자열
