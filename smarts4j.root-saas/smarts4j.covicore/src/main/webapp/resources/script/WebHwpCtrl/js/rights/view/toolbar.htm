도구 상자

도구 상자는 문서를 작성하면서 자주 사용되는 기능을 아이콘 모양으로 만들어 모아 놓은 곳입니다.

웹한글은 기본 도구 상자와 가장 자주 사용하는 기능을 모은 서식 도구 상자를 제공합니다. 편집 영역 확보 또는 작업 효율을 위해 도구 상자를 표시하거나 숨길 수 있습니다.

따라 하기

도구 상자 접기/펴기

  1. [보기-도구 상자-기본] 또는 [보기-도구 상자-서식]을 누릅니다. 기본 도구 상자 또는 서식 도구 상자가 선택 해제되고 화면에서 해당 도구 상자가 표시되지 않습니다.
  2. 도구 상자가 접힌 상태에서 [보기-도구 상자-기본] 또는 [보기-도구 상자-서식]을 누르면 선택한 도구 상자가 화면에 다시 표시됩니다.

참고

도구 상자 유형

기본 도구 상자 영역에는 문서 편집 시 가장 기본적으로 사용되는 메뉴가 제공되며, 서식 도구 상자에는 글자 또는 문단 서식 편집 시 자주 사용되는 메뉴가 포함되어 있습니다.
