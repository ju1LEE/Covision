캡션 넣기

본문에 들어가는 그림이나 표, 글상자, 그리기 개체, 수식은 필요에 따라 번호와 제목, 간단한 설명 등을 붙입니다. 이처럼 개체에 붙인 제목을 "캡션"(caption)이라고 합니다.

[입력-캡션 넣기]는 캡션 내용을 넣거나 고치는 기능입니다. 한 번 캡션을 넣어 두면 개체의 위치를 바꾸더라도 항상 캡션이 따라다니기 때문에 매우 편리합니다.

따라 하기

  1. [입력-그림]을 실행하여 그림을 삽입합니다.
  2. 캡션을 넣을 그림을 선택합니다.
  3. [입력-캡션 넣기]를 누른 다음 [오른쪽 위]를 선택합니다.
  4. 캡션 내용이 그림의 오른쪽 위에 나타납니다.

자세한 정보

[개체 속성] 대화 상자의 [여백/캡션] 탭

캡션

  • 위치
    캡션 내용이 들어갈 위치는 개체의 테두리 바깥쪽입니다.

    '위, 아래, 왼쪽/위, 왼쪽, 왼쪽/아래, 오른쪽/위, 오른쪽, 오른쪽/아래' 중에서 캡션의 위치를 정할 수 있습니다.

  • 크기
    캡션의 위치가 '위'나 '아래'일 때에는 캡션의 크기는 개체의 가로 크기와 같습니다. 하지만 캡션의 위치를 개체의 '왼쪽'이나 '오른쪽'으로 설정했을 때에는 캡션의 크기를 지정할 수 있습니다.
  • 개체와의 간격
    개체와 캡션 내용 사이의 간격을 얼마큼 띄울 것인지 정합니다. 처음 값은 3mm입니다. 한/글은 개체와의 간격에 음수 값을 지원하므로, 필요하다면 [개체와의 간격]에 음수 값을 입력하여 사용할 수 있습니다.
  • 여백 부분까지 너비 확대
    개체에 바깥 여백을 설정하더라도 캡션은 바깥 여백 부분을 제외하고 순수한 개체 영역부터 시작됩니다. 이 항목을 선택하면 왼쪽이나 오른쪽 바깥 여백부터 캡션을 시작합니다.
  • 한 줄로 입력
    이 항목을 선택하면 캡션 영역에 글을 입력하면서 커서가 오른쪽 경계선에 닿아도 커서가 다음 줄로 넘어가지 않고 한 줄에 모두 입력됩니다.

참고

캡션 지우기

캡션 영역에서 <Delete>나 <Backspace>를 눌러 내용을 모두 지우고 본문으로 빠져나온 경우, 내용은 비어 있지만 캡션 영역은 그대로 남아 있습니다. 따라서 캡션 영역을 마우스로 누르면 커서가 캡션 영역 안으로 들어갑니다.

캡션 영역에서 마우스 오른쪽 단추의 빠른 메뉴를 열어 [캡션 지우기]를 실행한 경우에는 캡션이 완전히 지워집니다.

캡션 이름 수정

캡션 영역을 마우스로 누르면 캡션 영역에 커서가 들어가므로, 내용을 수정할 수 있습니다.
