참고

다른 이름으로 저장

다른 편집 프로그램에서 생성했거나 한컴오피스에서 편집한 적이 없는 문서를 저장하려는 경우 다른 이름으로 저장 대화 상자가 나타나며 다른 이름으로 문서를 저장해야 합니다.

문서를 저장하는 동안 저장 상태에 대한 메시지는 편집 화면 오른쪽 상단에 표시됩니다.

자동 저장 환경일 때 저장 메뉴 지원 안 함

자동 저장 환경에서는 문서를 저장하지 않아도 문서가 자동으로 저장됩니다. 따라서 자동 저장 환경일 때는 저장 메뉴가 나타나지 않습니다. 저장 방식은 서버의 설정을 따르며 사용자 임의로 변경할 수 없습니다.
