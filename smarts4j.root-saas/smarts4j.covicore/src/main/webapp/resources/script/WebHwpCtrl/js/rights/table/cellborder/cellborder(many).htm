하나의 셀처럼 적용

셀 블록을 정형으로 설정한 다음 [셀 테두리/배경-하나의 셀처럼 적용]을 선택하면 셀 블록을 설정한 여러 셀을 하나의 셀처럼 처리합니다.

셀 블록을 설정한 다음 [셀 테두리/배경] 대화 상자에서 표의 각 셀을 에워싸는 테두리의 모양을 바꾸거나 대각선 긋기, 또는 셀마다 배경색과 배경 그림 등을 넣어 표를 보기 좋게 꾸밉니다.

참고

셀 블록

[셀 테두리/배경-하나의 셀처럼 적용] 기능은 셀 블록의 설정 상태가 정형일 때만 사용할 수 있습니다.

2개 이상의 셀을 비정형으로 셀 블록 설정을 하거나, 2개 이상의 셀을 비연속으로 셀 블록을 설정했을 때는 [하나의 셀처럼 적용] 기능은 사용할 수 없습니다.

테두리

셀 블록을 설정한 지정된 범위에 들어갈 셀 테두리의 선 종류와 굵기, 색을 선택합니다.

여러 개의 셀을 셀 블록으로 설정한 다음 [하나의 셀처럼 적용]을 선택하면, 여러 개의 셀 블록을 하나의 셀처럼 인식하여 셀 테두리를 설정할 수 있습니다.

배경

셀 블록을 설정한 지정된 범위에 들어갈 '색, 그림' 등 셀 배경을 설정합니다.

여러 개의 셀을 셀 블록으로 설정한 다음 [하나의 셀처럼 적용]을 선택하면, 여러 개의 셀 블록을 하나의 셀처럼 인식하여 셀 배경을 설정할 수 있습니다.

대각선

셀 블록을 설정한 지정된 범위에 들어갈 다양한 '대각선'을 넣을 수 있습니다.

여러 개의 셀을 셀 블록으로 설정한 다음 [하나의 셀처럼 적용]을 선택하면, 여러 개의 셀 블록을 하나의 셀처럼 인식하여 대각선을 설정할 수 있습니다.

셀 단위로 적용

셀 테두리/배경은 각 셀 단위로 적용됩니다. 셀 블록을 설정하고 셀 배경을 지정하더라도 모든 셀에 걸치는 배경이 적용되는 것이 아니라, 각 셀마다 지정한 배경이 적용됩니다.

단축키

셀 블록 상태에서 다음과 같은 단축키를 사용하여 셀 테두리/배경 작업을 쉽게 할 수 있습니다.

  • 셀 테두리: <L>
  • 셀 배경: <C>

관련 기능
