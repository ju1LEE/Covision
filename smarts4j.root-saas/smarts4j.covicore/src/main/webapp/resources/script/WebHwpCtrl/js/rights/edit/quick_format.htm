모양 복사 <Ctrl+Alt+C>

[모양 복사]는 커서 위치의 글자 모양이나 문단 모양 등을 다른 곳으로 간편하게 복사하는 기능입니다. 특정한 모양을 반복적으로 자주 지정해야 하는 경우에 매우 편하게 쓸 수 있습니다.

따라 하기

  1. 복사할 글자 모양을 가진 글자 바로 뒤에 커서를 놓고 [편집-모양 복사]를 실행합니다.

  2. [모양 복사] 대화 상자가 나타나면 [본문 모양 복사] 항목에서 [글자 모양]을 선택하고 [복사] 단추를 누릅니다.
  3. 모양을 바꿀 부분을 블록으로 설정합니다.

  4. [편집-모양 복사]를 다시 실행합니다.
  5. 블록으로 설정한 부분의 글자 모양이 복사해 두었던 글자 모양으로 바뀝니다.

자세한 정보

[모양 복사] 대화 상자

본문 모양 복사

  • 글자 모양

    커서가 있는 곳의 글자 모양을 복사합니다.

  • 문단 모양

    커서가 있는 곳의 문단 모양을 복사합니다.

  • 글자 모양과 문단 모양 둘 다 복사

    커서가 있는 곳의 글자 모양과 문단 모양 두 가지 모두를 복사합니다.

셀 모양 복사

  • 셀 속성

    커서가 놓여 있는 셀 위치의 글자 모양이나 글자 모양, 안 여백, 세로 정렬 상태, 제목 셀 속성, 한 줄로 입력 등의 셀 속성을 복사합니다.

  • 셀 테두리

    커서가 놓여 있는 셀의 테두리 종류와 굵기, 색깔, 대각선 종류, 적용 방향 등의 선 종류를 복사합니다.

  • 셀 배경

    커서가 놓여 있는 셀의 배경 채우기 속성을 그대로 복사합니다.

표에서 적용 대상

  • 본문 모양과 셀 모양 둘다

    [본문 모양 복사]에서 설정한 항목과 [셀 모양 복사]에서 설정한 항목을 모두 적용합니다.

  • 셀 모양만

    [본문 모양 복사]를 제외한 [셀 모양 복사]에서 설정한 항목만 모양 복사를 수행합니다.

참고

보통 상태와 블록 상태

[모양 복사]는 블록을 설정하지 않은 보통 상태에서는 커서 위치의 모양을 임시 저장소에 기억(복사)시키는 기능이 실행되고, 블록을 설정한 상태에서는 블록으로 설정한 부분에 기억된 모양을 덮어 쓰는 기능이 실행됩니다.

표 안에서의 모양 복사

표 안에서는 글자 모양과 문단 모양뿐만 아니라 현재 셀의 셀 속성과 선 종류, 셀 배경까지도 복사해 두었다가 다른 셀에 그대로 덮어 쓸 수 있습니다.

[셀 속성 복사, 셀 테두리 복사, 셀 배경 복사]는 표 안에서 [편집-모양 복사]를 실행한 경우에만 선택할 수 있습니다.

주의

커서 위치

글자 모양이나 글자 스타일을 복사할 때에는 커서 위치를 조심해야 합니다. 복사할 글자의 바로 뒤에 커서를 놓고 [편집-모양 복사]를 실행하십시오.
