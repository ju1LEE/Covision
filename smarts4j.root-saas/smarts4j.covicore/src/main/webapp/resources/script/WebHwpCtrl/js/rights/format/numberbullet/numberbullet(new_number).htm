시작 번호 방식

현재 문단의 문단 번호를 앞 번호 목록에 이어서 시작할 것인지 새 번호로 시작할 것인지 선택합니다.

따라 하기

  1. 시작 번호를 바꿀 번호 문단에 커서를 놓습니다.
  2. [서식-문단 번호 모양]를 실행합니다.
  3. [글머리표 및 문단 번호] 대화 상자가 나타나면, [문단 번호] 탭의 [시작 번호 방식-새 번호 목록 시작]을 선택합니다.
  4. 새 번호 입력 상자에 원하는 시작 번호를 입력합니다.
  5. [설정] 단추를 누릅니다. 현재 문단의 번호가 입력한 시작 번호로 바뀝니다.

자세한 정보

앞 번호 목록에 이어

현재 문단의 앞쪽으로 가장 가까운 번호 문단의 번호를 계속 이어 나갑니다.

이전 번호 목록에 이어

문단 번호가 매겨진 문단 사이에 다른 문단 번호 속성이 있는 문단을 끼워 넣어도 현재 위치의 문단 번호는 이전 번호 목록에 계속 이어집니다.

새 번호 목록 시작

현재 문단부터 새로운 번호로 다시 시작합니다. 적용할 새로운 문단 번호의 시작 값을 입력하십시오.

참고

새 번호를 적용한 이후 문단

[새 번호로 시작]을 하면 현재 문단 이후의 번호 문단이 [앞 번호 목록에 이어]의 속성을 가지고 있는 경우, 새 번호에 이어서 자동으로 번호를 다시 매겨줍니다.
