문단 부호

문서를 작성하면서 글자 입력 도중에 <Enter>를 누른 곳을 줄 바꿈 문자()로 화면에 표시해 주는 기능입니다. 보통 문단 끝 부분에서 <Enter>를 누르게 되는데, 이렇게 문단을 분리하는 <Enter>를 '하드 리턴(Hard Return)'이라고 부릅니다.

따라 하기

문단 부호 보이기/숨기기

  1. [보기-문단 부호]를 실행합니다.
  2. [문단 부호] 메뉴가 선택되면서 문단 부호 보이기 상태가 됩니다.

    또한 문서 창에는 <Enter>를 누른 곳마다 파란색의 줄 바꿈 문자()가 표시됩니다.

  3. 다시 [보기-문단 부호]를 실행합니다.
  4. [문단 부호] 메뉴가 선택 해제되면서 문단 부호 감추기 상태가 됩니다.

참고

문단 부호 인쇄

문단 부호는 편집 화면에서만 보일 뿐 문서를 인쇄했을 때에는 나타나지 않습니다.

문단 부호 색깔

문단 부호는 파란색으로 나타나며, 사용자가 이 표시의 색깔과 모양을 바꿀 수는 없습니다.

강제 줄 바꿈

줄 끝에서 <Shift+Enter>를 누르면 문단은 분리하지 않고 강제로 줄만 바꿔줍니다.

<Shift+Enter>를 눌러 강제 줄 바꿈을 수행한 곳에는 파란색의 강제 줄 바꿈 문자()가 화면에 표시됩니다.

조판 부호와 문단 부호 보이기/숨기기

문단 부호 보이기/숨기기 설정 상태는 조판 부호 보이기/숨기기의 영향을 받지 않습니다.

즉, 문단 부호 보이기 상태에서 조판 부호 보이기를 설정했다가 다시 조판 부호 감추기를 설정하더라도 문단 부호는 보이기 상태가 됩니다. 또한 문단 부호 감추기를 설정한 상태에서 조판 부호 보이기를 설정했다가 조판 부호 감추기를 설정하면, 문단 부호는 감추기 상태가 그대로 유지됩니다.
