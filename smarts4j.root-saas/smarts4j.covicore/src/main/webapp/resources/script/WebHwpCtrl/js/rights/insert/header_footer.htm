머리말/꼬리말

제목이나 소제목을 모든 쪽의 맨 위나 맨 아래 동일한 위치에 반복하여 나타내는 것을 머리말/꼬리말이라고 합니다. 모든 문서에 같은 내용을 위쪽이나 아래쪽에 넣을 때 사용하면 편리합니다. 머리말은 문서의 위쪽 여백과 본문 사이에 꼬리말은 본문과 아래쪽 여백 사이에 삽입됩니다. 머리말과 꼬리말을 편집하는 상태에서는 본문의 내용을 편집하거나 선택할 수 없습니다.

따라 하기

머리말 또는 꼬리말 삽입

머리말 또는 꼬리말을 삽입하려면 다음을 실행합니다.

  1. 쪽-머리말 또는 꼬리말을 누릅니다.
  2. 머리말 또는 꼬리말을 추가할 쪽 유형을 선택합니다. 편집 화면에 머리말 또는 꼬리말 영역이 점선 테두리로 나타납니다.
  3. 점선 테두리 안의 영역에 내용을 입력합니다. 필요에 따라 그림, 표 등을 삽입할 수 있습니다.
  4. 머리말/꼬리말 영역 이외의 임의의 지점을 누릅니다. 점선 테두리 영역이 없어지면서 입력한 내용이 본문 내용보다 흐리게 나타납니다.

참고

빠른 편집

머리말/꼬리말을 보다 신속하게 입력하려면 문서에서 머리말/꼬리말 영역을 누릅니다.

쪽 번호 입력

문서에 쪽 번호를 입력하려면 머리말 또는 꼬리말 하위의 쪽 유형을 선택합니다.
