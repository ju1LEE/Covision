글상자

커다란 제목을 넣거나 본문 중간에 박스형 요약 글을 넣을 때 주로 [글상자]를 이용합니다.

글상자는 위치와 크기 조절, 글상자 안의 채우기 효과, 테두리의 모양과 색깔 바꾸기 등을 자유롭게 설정할 수 있어 쓰임새가 다양합니다.

따라 하기

  1. [입력-글상자]를 실행합니다.
  2. 사각형의 글상자가 나타나고, 그 안에서 커서가 깜박입니다.

  3. "웹한글은 워드 프로세서의 새로운 지평입니다."라고 입력합니다.

    글상자의 크기보다 많은 내용을 입력하면 글상자의 높이가 자동으로 늘어납니다.

  4. 글상자 내용을 블록으로 설정한 다음 [서식-글자 모양] 대화 상자에서 글꼴과 글자 크기를 조정합니다.

참고

글상자에서 나오기

글상자 안에 내용 입력을 마친 다음, 마우스 포인터를 글상자 바깥의 편집 영역 아무 곳에나 놓고, 마우스 왼쪽 단추를 한 번 누르면 글상자에서 빠져나옵니다.

개체 속성 변경하기

입력된 개체의 속성을 변경하려면 [서식-개체 속성]을 실행하거나 개체의 테두리 부분을 마우스로 두 번 눌러 [개체 속성] 대화 상자를 불러옵니다. [개체 속성] 대화 상자를 이용하여 개체의 크기와 위치 조절, 개체의 테두리 모양과 채우기 효과 등을 변경할 수 있습니다.
