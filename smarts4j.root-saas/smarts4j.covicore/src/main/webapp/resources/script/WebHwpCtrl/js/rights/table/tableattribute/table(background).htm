표/셀 속성: 배경

[표/셀 속성] 대화 상자의 [배경] 탭에서는 표의 배경을 색깔과 무늬 또는 그림으로 채우기 등을 지정하여 다양한 채우기 효과를 낼 수 있습니다.

자세한 정보

[표/셀 속성] 대화 상자의 [배경] 탭

색 채우기 없음

[색]이나 [그림] 채우기 효과를 적용하지 않을 경우에는 [색 채우기 없음]을 선택합니다.

색
  • 면 색

  • 색상표를 눌러 색상 팔레트가 나타나면 배경 색을 선택합니다.

  • 무늬 색

  • [무늬 모양]을 선택한 경우 [무늬 색] 항목에서 색상표를 눌러 색상 팔레트가 나타나면 배경을 채울 무늬 색을 선택합니다.

  • 무늬 모양

  • [무늬 모양] 목록에서 배경을 채울 무늬 모양을 선택합니다.

그림

그림 파일을 불러와 배경을 그림으로 채웁니다.

  • 그림 파일

  • [파일 선택] 또는 [찾아보기] 단추를 눌러 배경을 채울 그림 파일을 불러옵니다.

  • 채우기 유형

  • 배경을 그림으로 채울 때, 바둑판식으로 채울 것인지, 크기에 맞추어 채울 것인지, 쪽 가운데에 넣을 것인지 목록에서 채우기 유형을 선택합니다.

  • 그림 효과

  • 불러온 그림에 대해 회색조 처리를 하거나, 흑백 효과 등을 줄 수 있습니다.

  • 밝기

  • 선택한 그림 파일의 밝기를 조절합니다. 밝기의 단계는 -100부터 100까지 설정할 수 있습니다.

  • 대비

  • 선택한 그림 파일의 명암을 조절합니다. 명암의 단계는 -100부터 100까지 설정할 수 있습니다.

  • 워터마크 효과

  • 불러온 그림 파일에 70%의 밝기와 -50%의 명암을 주어 원래 그림을 밝고 명암 대비가 작은 그림으로 바꿉니다.
