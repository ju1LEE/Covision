개체 속성

본문에 삽입한 개체의 크기, 위치, 여백 등과 같은 속성을 [개체 속성] 대화 상자의 [기본], [여백/캡션], [선], [채우기] 탭에서 각각 설정할 수 있습니다.

따라 하기

  1. [입력-그림]을 실행한 다음 [그림 넣기] 대화 상자가 나타나면 그림 파일의 위치와 파일 이름을 선택하고 [넣기] 단추를 누릅니다.
  2. 편집 화면에 그림이 삽입되면 그림을 선택한 상태에서 <P>를 누르거나 [서식-개체 속성]을 실행합니다.
  3. [개체 속성] 대화 상자가 나타나면 [기본], [여백/캡션], [선] 탭 중 원하는 탭을 선택합니다.
  4. 원하는 속성을 변경한 다음 [설정] 단추를 누릅니다.
