표 만들기

문서를 만들면서 복잡한 내용이나 수치 자료를 일목요연하게 정리하고자 할 때에는 표를 이용합니다.

한/글의 [표 만들기]는 표 안에 다른 표를 만들어 넣을 수 있으며, 한 쪽을 넘어가는 표를 자동으로 나누어 주기 때문에 매우 편리합니다.

따라 하기

  1. [표-표 만들기]를 실행합니다.
  2. [표 만들기] 대화 상자가 나타나면 줄 수와 칸 수를 지정하고 [만들기] 단추를 누릅니다.
  3. 표가 삽입되며 커서는 표의 첫째 줄 첫째 칸에 놓여 있습니다.
  4. 이 상태에서 마치 본문을 편집하듯이 표 안에 내용을 입력합니다.
  5. 글자판의 <Shift+Esc>를 누르거나 마우스로 표 바깥 부분을 한 번 누르면 표에서 빠져나옵니다.

자세한 정보

[표 만들기] 대화 상자

줄/칸

표의 줄 수와 칸 수를 지정합니다. 표를 만들 때 지정할 수는 있는 최대 줄 수와 칸 수는 64X64입니다.

크기 지정

표의 크기를 직접 입력할 수 있습니다.

  • 너비
    • 단에 맞춤

      커서가 있는 단의 폭에 꽉 차는 너비로 표를 만듭니다.

    • 문단에 맞춤

      커서가 있는 문단 폭에 꽉 차는 너비로 표를 만듭니다.

    • 임의 값

      원하는 너비 값을 직접 입력하여 표를 만듭니다.

  • 높이
    • 자동

      셀의 기본 높이에 줄 수를 곱한 크기로 표를 만듭니다.

    • 임의 값

      원하는 높이 값을 직접 입력하여 표를 만듭니다.

기타

표를 만들 때 표의 위치를 글자처럼 취급할지 여부를 지정합니다.

  • 글자처럼 취급

    표를 보통 글자와 같게 취급합니다. 따라서 글을 입력하거나 지우는 대로 표의 위치가 같이 변합니다.

참고

표 안에 표 만들기

표 안에서 [입력-표-표 만들기]를 실행하여 새로운 표를 만들고 편집할 수 있습니다.

표 안에 삽입한 표가 현재 표의 셀보다 클 경우 나머지 부분은 가려져 보이지 않습니다. 그러므로 표 안에 삽입한 표의 크기에 맞게 셀 크기를 조정하십시오.

최소 표 크기

너비/높이를 [임의 값]으로 지정하여 표를 만들면서 줄 수와 칸 수에 비해 너무 작은 값을 지정하면, 최소 셀 너비/높이에 줄 수와 칸 수를 곱한 크기로 표를 만들어 줍니다.

표 겹쳐 놓기

두 개의 표가 [본문과의 배치-자리 차지]로 설정되어 있을 경우에는 표끼리 겹쳐지지 않습니다. 이때는 표를 선택한 후 [표/셀 속성]의 [기본] 탭에서 [위치-서로 겹침 허용] 항목을 선택해야 표를 겹쳐 놓을 수 있습니다.
