지우기 <Ctrl+E>

블록으로 선택한 내용을 지웁니다. 문서에서 필요 없는 부분을 한꺼번에 지울 때 사용합니다.

따라 하기

  1. 문서에서 지우려는 내용을 블록으로 선택합니다.
  2. [편집-지우기]를 실행합니다. 블록으로 선택한 내용이 지워집니다.

참고

지우기 유의 사항

  • 지운 내용은 기억되지 않음

    [지우기]를 실행한 것은 이름 그대로 지운 것일 뿐입니다. [복사하기], [오려 두기]와 달리 그 내용이 클립보드에 저장되지 않습니다.

  • 지우기와 되돌리기

    지금 [지우기]를 실행한 것이 실수였다면, 바로 [되돌리기]를 실행해야 합니다. 바로 전에 실행한 [지우기]의 동작을 취소하므로 지운 내용이 되살아납니다.

  • <Delete>와 <BackSpace>

    [지우기]의 단축키는 <Ctrl+E>입니다. 블록을 설정한 상태에서 <Delete>나 <BackSpace>를 눌러도 [지우기]와 똑같이 동작합니다.

  • 블록을 설정한 상태에서 입력하면 블록 내용이 지워짐

    블록을 설정한 상태에서 글자판을 눌러 입력을 시작하면, 블록으로 설정된 부분이 지워지면서 그 자리에 새로운 내용이 입력됩니다.
