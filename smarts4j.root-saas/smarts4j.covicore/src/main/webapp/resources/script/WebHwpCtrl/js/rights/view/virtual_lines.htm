투명 선

표에서 각각의 선 종류를 선택할 수 있습니다. 이때 선 종류를 [선 없음]으로 지정하게 되면 편집할 때나 인쇄할 때 선이 보이지 않습니다. 이런 경우 [보기-투명 선]을 선택해 놓으면, [선 없음]으로 선택한 곳이 빨간색 점선으로 나타나기 때문에 편집하는 데 도움이 됩니다.

따라 하기

투명 선 보이기/숨기기

  1. [보기-투명 선]을 실행합니다.
  2. [투명 선] 메뉴가 선택되면서 투명 선 보이기 상태가 됩니다.

    또한 표에서 선 종류를 [선 없음]으로 지정한 부분은 빨간색의 점선으로 나타납니다.

  3. 다시 [보기-투명 선]을 실행합니다.
  4. [투명 선] 메뉴가 선택 해제되면서 선 종류를 [선 없음]으로 지정한 부분이 실제 모양대로 보이지 않게 됩니다.

참고

투명 선 인쇄

[투명 선] 보이기를 설정했을 때 나타나는 선은 편집 화면에서만 보일 뿐 인쇄했을 때에는 나타나지 않습니다.

투명 선 색깔

[투명 선] 보이기를 설정했을 때 나타나는 선은 항상 빨간색의 점선으로 표시되며, 사용자가 이 선의 색깔과 모양을 바꿀 수는 없습니다.
