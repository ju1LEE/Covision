셀 테두리/배경: 테두리 <L>

[셀 테두리/배경] 대화 상자의 [테두리] 탭에서는 표의 각 셀마다 다양한 모양의 셀 테두리선을 넣을 수 있으며, 원하는 방향의 테두리만을 보이거나 감출 수 있습니다.

[테두리] 탭의 [적용 범위] 항목은 [표-셀 테두리/배경-각 셀마다 적용]을 실행한 경우에만 나타나는 항목이며, 현재 표의 셀 블록 설정 상태에 따라 [모든 셀, 선택된 셀]을 범위로 제시합니다.

따라 하기

  1. 표 안에 커서를 놓고 [표-셀 테두리/배경-각 셀마다 적용]을 실행합니다.
  2. [셀 테두리/배경] 대화 상자가 나타나면 [테두리] 탭을 선택합니다.
  3. [테두리 종류] 목록 단추를 눌러 맨 마지막에 있는 삼중선을 선택합니다.
  4. [선 굵기]는 자동으로 2mm가 설정됩니다.
  5. 적용 영역에서 [바깥쪽] 아이콘을 눌러 선택한 테두리를 셀의 바깥쪽 방향에 적용합니다.
  6. [설정] 단추를 누르면 표의 바깥쪽 테두리가 삼중선으로 바뀝니다.

자세한 정보

[셀 테두리/배경] 대화 상자의 [테두리] 탭

테두리

셀 테두리는 11가지의 선 [종류] 중에서 선택할 수 있으며, 각각의 테두리 종류에 대하여 [굵기]와 [색]을 지정할 수 있습니다.

  • 종류

  • 11가지 선 목록에서 원하는 종류를 선택합니다. 각각의 선 종류에 대하여 색과 두께를 정할 수 있습니다.

  • 굵기

  • 선 굵기는 목록에서 선택만 할 수 있고, 값을 직접 입력해서 굵기를 지정할 수는 없습니다.

  • 색

  • 색상표를 누르면 나타나는 색상 팔레트에서 테두리 색을 선택합니다.

적용 위치

테두리 종류, 굵기, 색을 지정한 다음 테두리 적용할 위치에 대한 아이콘을 누르면 해당 위치에 테두리가 적용됩니다.

적용 범위

[표-셀/테두리 배경-각 셀마다 적용]을 실행한 경우에만 나타나며, [표-셀/테두리 배경-하나의 셀처럼 적용]을 실행한 경우에는 나타나지 않습니다.

  • 모든 셀

  • 현재 표의 모든 셀에 대하여 셀 테두리/배경 속성을 똑같이 적용합니다.

  • 선택된 셀

  • 셀 블록을 설정한 상태에서는 [선택된 셀]이 처음 값으로 제시되며, 선택된 셀에만 셀 테두리/배경 속성을 적용합니다.
