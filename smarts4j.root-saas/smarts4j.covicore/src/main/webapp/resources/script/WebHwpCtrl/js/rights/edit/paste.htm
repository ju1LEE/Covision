붙이기 <Ctrl+V>

[복사하기]나 [오려두기]를 실행하여 클립보드에 임시로 저장한 내용을 커서 위치에 다시 붙여 넣습니다.
클립보드에 저장된 내용은 여러 번 반복하여 [붙이기]를 수행할 수 있습니다.

따라 하기

  1. 잘라내거나 복사하려는 내용을 블록으로 선택합니다.
  2. <Ctrl+C> 또는 <Ctrl+X>를 누릅니다.
  3. 커서를 원하는 위치로 옮겨 [편집-붙이기] 또는 <Ctrl+V>를 누르면 블록으로 선택한 내용이 커서 위치에 붙습니다.

참고

단축키 사용

일부 웹 환경에서 클립보드에 접근하려면 반드시 단축키를 사용해야 합니다. 따라서 오려 두기, 복사하기, 붙이기 기능을 실행하려면 다음 단축키를 사용하시기 바랍니다. Internet Explorer에서는 단축키 대신 [편집-붙이기]를 사용할 수 있습니다.

  • 오려 두기: Ctrl + X
  • 복사하기: Ctrl + C
  • 붙이기: Ctrl + V

브라우저의 다른 탭 또는 다른 창으로 붙이기

한 문서에서 복사하거나 오려 둔 내용을 다른 탭 또는 창에도 붙일 수 있습니다.
어느 문서의 내용 일부가 다른 곳에서도 그대로 필요할 때, 원하는 내용을 블록으로 선택하고 <Ctrl+C>를 누른 다음 커서를 다른 탭 또는 다른 창으로 옮겨 <Ctrl+V>를 실행합니다.

다른 응용 프로그램에 붙이기

클립보드에 저장된 내용은 한/글 프로그램 안에서는 물론 다른 응용 프로그램에서도 같은 방법으로 불러 쓸 수 있습니다.
다만 그림이나 글자 모양, 문단 모양 등 세부적인 사항은 옮겨지는 다른 응용 프로그램이 지원하는 정도에 따라 달라질 수 있습니다.

표 안에서의 붙이기

셀 블록을 설정하고 [복사하기]를 하면 표 안의 내용뿐만 아니라 블록으로 설정한 표의 틀까지 복사됩니다.

표 안에서 [붙이기]를 하면 [셀 붙이기] 대화 상자가 나타납니다. 여기서 복사한 표를 끼워 넣을 것인지, 덮어 쓸 것인지, 셀 안에 표로 넣을 것인지 붙일 방법을 선택합니다.

활용

대화 상자의 입력 상자에서도 사용

[복사하기], [오려두기], [붙이기]는 대화 상자에서 [파일 이름]과 같이 사용자가 내용을 직접 입력할 수 있는 대부분의 입력 상자에서도 쓸 수 있습니다.
