블록 평균 <Ctrl+Shift+A>

빈 셀을 포함하여 줄이나 칸을 셀 블록으로 설정하고 [블록 평균]을 실행하면, 평균을 계산하여 오른쪽이나 아래의 빈 셀에 계산 결과를 자동으로 입력하여 줍니다.

따라 하기

  1. 표에서 블록 계산할 셀들을 셀 블록을 설정합니다.

  2. [표-블록 계산식-블록 평균]을 실행합니다.
  3. 평균을 계산하여 오른쪽이나 아래의 빈 셀에 계산 결과를 자동으로 입력하여 줍니다. 이때 계산되는 평균은 소수점 둘째 자리까지 계산됩니다.

참고

바로 앞의 계산 결과는 무시

블록 평균 결과를 입력할 셀의 바로 앞 셀, 즉 [블록 평균]에서 사용한 셀의 마지막 셀에 [계산식] 결과가 입력되어 있다면, 이 계산 결과는 블록 평균에 포함하지 않습니다. 이것은 아래의 예제 표와 같이, "총점"을 [계산식]으로 구한 다음 바로 뒤 셀의 "평균" 역시 [계산식]을 사용하는 경우가 많기 때문입니다.
