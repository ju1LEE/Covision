줄/칸 추가하기

표를 만들다 보면 새로운 줄이나 칸이 더 필요한 경우가 생깁니다. 이런 경우, [표-줄/칸 추가하기]를 이용해 줄이나 칸을 더 삽입할 수 있습니다.

따라 하기

  1. 줄 또는 칸을 더 추가하고 싶은 셀에 커서를 놓습니다.
  2. [표-줄/칸 추가하기]를 누른 다음 하위 메뉴에서 줄 또는 칸을 추가할 위치를 선택합니다.
  3. 추가하려는 줄 또는 칸의 개수만큼 위 과정을 반복합니다.

참고

셀 블록

셀 블록을 설정해 [줄/칸 추가하기]를 수행할 때에는 반드시 하나의 셀만 셀 블록으로 설정해야 합니다. 2칸 이상의 셀을 셀 블록으로 설정하면 [줄/칸 추가하기]를 할 수 없습니다.

<Tab>과 <Ctrl+Enter>

표에서 셀과 셀 사이를 이동할 때에는 <Tab>을 누릅니다. 그런데 표의 마지막 셀에서 <Tab>을 누르면 아래쪽에 줄이 추가됩니다.

또한 표 안에서 <Ctrl+Enter>를 눌러도 현재 줄의 아래쪽에 줄이 추가됩니다.
