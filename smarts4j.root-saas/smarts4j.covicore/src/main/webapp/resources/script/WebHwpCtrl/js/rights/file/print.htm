인쇄

현재 편집 화면에 있는 문서를 프린터로 인쇄합니다.

문서 전체나 일부를 여러 매 인쇄할 수 있고, 인쇄물의 용도에 따른 확대/축소 등 다양한 인쇄 조건을 지정할 수 있습니다.

인쇄 기능을 실행하면 문서가 pdf 문서로 자동 변환되며 pdf로 변환된 문서는 C\Users\Downloads 폴더에 저장됩니다.

문서 인쇄

편집 중인 문서를 인쇄하려면 다음을 실행합니다.

  1. 파일-인쇄를 누릅니다.
  2. PDF 문서 변환이 완료되면 PDF 문서를 바로 열거나 로컬 저장소에 저장합니다.
    단, Chrome 브라우저를 사용하는 경우 별도의 사용자 동작 없이 PDF 문서를 편집 화면에서 미리 볼 수 있습니다.
  3. 열려 있는 PDF 문서에서 인쇄를 실행하거나 로컬 저장소에 저장된 PDF 문서를 열어 인쇄를 실행합니다.
