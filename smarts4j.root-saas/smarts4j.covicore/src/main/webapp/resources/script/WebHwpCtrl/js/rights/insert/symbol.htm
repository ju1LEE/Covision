문자표

문자표를 이용하여 다양한 문자나 기호를 입력할 수 있습니다. 글자판에서 입력하기 어려운 각종 기호나 문자도 문자표를 이용하면 쉽게 입력할 수 있습니다.

따라 하기

특수 문자 삽입

문서에 특수 문자를 삽입하려면 다음을 실행합니다.

  1. [입력-문자표]를 누릅니다.
  2. [문자표] 대화 상자의 원하는 문자 또는 기호의 유형을 선택합니다.
  3. 문서에 입력할 문자 또는 기호를 누릅니다.
  4. [넣기] 단추를 누릅니다.

자세한 정보

[문자표] 대화 상자

문자 유형 목록

목록 펼침 단추()를 눌러 원하는 문자가 포함된 유형을 선택합니다.

문자 선택 영역

선택한 문자 유형에 따른 문자 목록을 보여줍니다. 여기에서 원하는 문자를 눌러서 선택합니다.

최근 사용한 문자

최근 사용한 문자가 표시됩니다.

입력 문자

문자 선택 영역에서 원하는 문자를 마우스로 누르면 [입력 문자]에 해당 문자가 표시됩니다.

[입력 문자]에 여러 개의 문자를 추가한 다음 [넣기] 단추를 누르면 [입력 문자]에 표시된 모든 문자가 편집 창에 삽입됩니다.

[넣기] 단추

[입력 문자]에 문자를 추가한 다음 [넣기] 단추를 누르면 [입력 문자]에 표시된 모든 문자가 편집 창에 삽입됩니다.
