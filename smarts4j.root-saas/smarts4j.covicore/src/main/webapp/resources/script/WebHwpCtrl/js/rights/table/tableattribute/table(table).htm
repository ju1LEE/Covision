여러 쪽 지원
  • 쪽 경계에서

  • 표가 여러 쪽에 걸칠 때 쪽 경계에서 어떻게 나눌 것인지 정합니다.

    아이콘 선택 항목 설명
    셀 단위로 나눔 쪽 경계에 걸리는 셀은 셀 전체를 다음 쪽으로 넘깁니다.
    나누지 않음 표가 여러 쪽에 걸치지 못하고 한 쪽에만 표시됩니다. 쪽 크기를 벗어나는 표 내용은 화면과 인쇄에 나타나지 않습니다.
  • 제목 줄 자동 반복

  • 여러 쪽에 걸치는 표에 대하여 [제목 줄 자동 반복]을 선택해 놓으면, 각 쪽의 표 시작마다 제일 위에 그 표의 모든 제목 줄들을 자동으로 넣어 줍니다.

    단, 표의 첫 번째 줄에서 [셀] 탭의 [제목 셀]을 선택해 놓아야만 [표] 탭의 [제목 줄 자동 반복]이 동작합니다.

  • 자동으로 나뉜 표의 경계선 설정

  • 쪽 경계선에 걸린 표가 자동으로 나눠지게 되었을 때 쪽 경계선에 표시할 선 종류, 굵기, 선 색을 지정할 수 있습니다.
