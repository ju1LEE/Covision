셀 합치기 <M>

[셀 합치기]는 셀 블록으로 설정한 두 개 이상의 셀(cell)을 하나로 합치는 기능입니다.

따라 하기

  1. 합치고자 하는 셀들을 셀 블록으로 설정합니다.
  2. [표-셀 합치기]를 실행합니다.
  3. 셀 블록으로 설정된 부분이 하나의 셀로 합쳐집니다.

참고

직사각형 형태의 셀 블록

셀 합치기를 할 때, 선택된 셀 블록의 형태는 반드시 직사각형이어야 합니다. 만약 불연속적인 셀 블록을 설정했거나 직사각형 모양이 아닌 셀 블록을 설정했을 때에는 [셀 합치기]를 할 수 없습니다.

합쳐진 셀의 셀 모양

[셀 합치기]를 하여 하나로 합쳐진 셀의 셀 배경과 선 종류, 대각선 종류 등은 셀 블록으로 설정했던 셀 중에서 왼쪽 맨 위의 첫 번째 셀의 셀 모양을 따라갑니다.

관련 기능

표 만들기, 셀 나누기, 셀 테두리/배경
