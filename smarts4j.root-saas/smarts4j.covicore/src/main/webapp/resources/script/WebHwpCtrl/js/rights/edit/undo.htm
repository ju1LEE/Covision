되돌리기 <Ctrl+Z>

조금 전에 실행한 명령을 취소하거나, 입력한 내용을 지우거나, 지운 내용을 되살리는 등 문서 편집 과정에서 수행한 동작을 다시 원래대로 되돌립니다.

사용자가 행한 동작을 단계별로 기억하고 있으므로, 되돌리기를 반복 실행하면 편집된 문서를 이전 단계로 계속 되돌릴 수 있습니다.

[되돌리기]는 문서 편집 작업을 하면서 발생하는 사용자의 실수를 최소한으로 줄여 주는 편리한 기능입니다.

따라 하기

  1. 현재 편집 화면에 있는 문서에서 글을 입력하거나, 글자 모양을 바꾸는 등의 편집 작업을 합니다.
  2. [편집-되돌리기]를 실행합니다.
  3. 가장 마지막에 했던 편집 동작이 취소되는 것을 편집 화면에서 확인할 수 있습니다.
  4. [편집-다시 실행]을 실행합니다.
  5. 조금 전에 [되돌리기]에 의하여 취소되었던 동작이 다시 실행된 것을 확인할 수 있습니다.

참고

되돌리기와 다시 실행

[되돌리기]를 실행하여 취소했던 명령이나 동작을 다시 실행하려면 [편집-다시 실행]을 실행합니다.

되돌리기 연속 실행

[되돌리기] 되는 내용은 사용자가 현재 문서에서 입력이나 편집에 관련된 동작을 한 것이 단계적으로 기억되는 것입니다. 따라서 연이어 실행하면 지금까지 작업한 과정을 거슬러 올라가며 취소하는 것과 같은 결과가 됩니다.

되돌릴 내용 없음

빈 문서를 새로 만들었을 때나 [불러오기]를 하여 문서를 연 직후에는 문서를 편집한 내용이 없으므로 [편집-되돌리기]를 실행할 내용도 없는 상태입니다.

이때 메뉴나 도구 상자에서 [되돌리기]는 흐리게 표시되며 실행할 수 없습니다.

되돌릴 내용 비우기

문서 편집을 하는 동안 되돌리기를 할 내용은 차례로 기억됩니다. 문서 편집을 끝내고 [문서 닫기]를 수행하면 되돌리기에 기억된 내용을 모두 지웁니다.

되돌리기의 범위

[되돌리기]를 실행할 동작으로 기억되는 것은 현재 편집 문서 안에서 실행된 입력과 편집 내용입니다.

여러 개의 문서를 열어 놓고 작업할 때에도 [되돌리기]할 내용은 각 문서마다 따로 기억됩니다.

또한 [확대/축소] 또는 [조판 부호]와 같이 현재 문서의 편집된 내용을 고치는 일과 상관없는 기능은 [되돌리기]에 해당하지 않습니다.
