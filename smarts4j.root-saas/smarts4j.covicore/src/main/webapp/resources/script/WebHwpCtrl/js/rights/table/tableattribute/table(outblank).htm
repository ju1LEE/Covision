  • 오른쪽

  • 개체 가장자리의 오른쪽과 본문 사이의 간격을 조정합니다.

캡션
  • 위치

  • 캡션 내용이 들어갈 위치는 개체의 테두리 바깥쪽입니다.
    캡션의 위치를 정할 수 있습니다.

  • 크기

  • 캡션의 위치가 [위]나 [아래]일 때에는 캡션의 크기는 개체의 가로 크기와 같습니다. 하지만 캡션의 위치를 개체의 [왼쪽]이나 [오른쪽]으로 설정했을 때에는 캡션의 크기를 지정할 수 있습니다.

  • 개체와의 간격

  • 개체와 캡션 내용 사이의 간격을 얼마큼 띄울 것인지 정합니다. 처음 값은 3mm입니다. 한/글에서는 개체와의 간격에 음수 값을 지원하므로, 필요한 경우 [개체와의 간격]에 음수 값을 입력하여 사용할 수 있습니다.

  • 여백 부분까지 너비 확대

  • 개체에 바깥 여백을 설정하더라도 캡션은 바깥 여백 부분을 제외하고 순수한 개체 영역부터 시작됩니다. 이 항목을 선택해 놓으면, 왼쪽이나 오른쪽 바깥 여백부터 캡션을 시작합니다.

  • 한 줄로 입력

  • 이 항목을 선택해 놓으면 캡션 영역에 글을 입력하면서 커서가 오른쪽 경계선에 닿아도 커서가 다음 줄로 넘어가지 않고 한 줄에 모두 입력됩니다.

참고

표와의 관계

바깥 여백은 표와 본문 글자 사이의 간격을 정하는 것입니다. 따라서 [글자처럼 취급]하지 않은 표와 표 사이에는 아무런 영향을 주지 않습니다.

본문과의 배치

바깥 여백은 [글자처럼 취급]한 표나 본문과의 배치를 [어울림] 또는 [자리 차지]로 지정한 표에 대해서만 유효합니다. [글 앞으로/글 뒤로] 배치한 표의 바깥 여백은 의미가 없습니다.

바깥 여백과 캡션

표에 바깥 여백을 설정하더라도 캡션은 바깥 여백 부분을 제외하고 순수한 개체 영역부터 시작됩니다. [캡션-여백 부분까지 너비 확대]를 선택해 놓으면, 왼쪽이나 오른쪽 바깥 여백부터 캡션을 시작합니다.
