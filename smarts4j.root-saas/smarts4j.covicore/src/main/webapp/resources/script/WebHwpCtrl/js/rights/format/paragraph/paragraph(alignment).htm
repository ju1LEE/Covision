문단 정렬 방식

문단 정렬 방식에 따라 줄 바꿈 기준과 위치 정렬 기준이 달라집니다.

따라 하기

  1. 정렬 방식을 지정할 문단에 커서를 놓고, [서식-문단 모양]을 실행합니다.
  2. [문단 모양] 대화 상자가 나타나면 [기본] 탭의 [정렬 방식]에서 [가운데 정렬] 아이콘()을 선택합니다.
  3. [설정] 단추를 누르면 글자를 문단의 가운데로 모읍니다.

자세한 정보

정렬 방식

아이콘 이름 설명 적용 예
양쪽 정렬 양쪽을 가지런하게 맞춥니다. 가장 널리 쓰이는 정렬 방식입니다.
왼쪽 정렬 왼쪽을 가지런하게 맞추고, 어절 단위로 줄 바꿈을 수행합니다.
가운데 정렬 문단을 가운데로 모으고, 어절 단위로 줄 바꿈을 수행합니다.
오른쪽 정렬 오른쪽을 가지런하게 맞추고, 어절 단위로 줄 바꿈을 수행합니다.
배분 정렬 글자 수에 상관없이 양쪽 정렬을 하되, 글자 사이를 일정하게 띄우는 정렬 방식입니다.
나눔 정렬 글자 수에 상관없이 양쪽 정렬을 하되, 어절 사이를 일정하게 띄우는 정렬 방식입니다.

참고

줄 바꿈 기준

문단 안에서 각 줄의 마지막에 한글이 올 때 줄 바꿈 기준을 '글자'나 '어절' 단위로 정렬할 수 있으며, 영문은 '단어'나 '하이픈', '글자' 단위로 정렬할 수 있습니다.

[왼쪽 정렬], [오른쪽 정렬], [가운데 정렬]을 선택하면 한글의 줄 바꿈 기준을 자동으로 '어절'로 바꾸어 줍니다.

한글과 영문이 섞여 있는 문장에서 한글은 '글자' 단위로, 영문은 '단어' 단위로 정렬하는 것이 가장 좋습니다.

단축키

문단 정렬 방식 중에서 양쪽 정렬, 왼쪽 정렬, 오른쪽 정렬, 가운데 정렬, 배분 정렬, 나눔 정렬 기능에 단축키가 배당되어 있어, 문서 편집 작업을 빠르게 수행할 수 있습니다.

단축키 기능
Ctrl+Shift+M 양쪽 정렬
Ctrl+Shift+L 왼쪽 정렬
Ctrl+Shift+R 오른쪽 정렬
Ctrl+Shift+C 가운데 정렬
Ctrl+Shift+T 배분 정렬
