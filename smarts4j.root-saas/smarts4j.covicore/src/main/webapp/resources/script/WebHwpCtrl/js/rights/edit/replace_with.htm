찾아 바꾸기 <Ctrl+H>

현재 편집하고 있는 문서에서 특정한 낱말(문자열)을 찾아서 다른 것으로 바꿉니다.

따라 하기

  1. [편집-찾아 바꾸기]를 실행합니다.
  2. [찾아 바꾸기] 대화 상자가 나타나면 [찾을 내용]에 현재 문서의 내용 중에 찾고자 하는 문자열을 입력합니다.
  3. [바꿀 내용]에는 [찾을 내용] 자리에 바꾸어 써넣을 문자열을 입력합니다.
  4. [바꾸기] 단추를 누르거나 <Enter>를 누릅니다.
  5. 본문에서 [찾을 내용]에 해당하는 낱말을 찾아 블록으로 감싸 보여 줍니다.
  6. 찾은 내용을 바꾸려면 [바꾸기] 단추를 누르고, 찾은 내용을 바꾸지 않고 그냥 지나가려면 [다음 찾기] 단추를 누릅니다.
  7. 찾아 바꾸기를 모두 완료했으면 [닫기] 단추를 눌러 [찾아 바꾸기] 대화 상자를 닫습니다.

자세한 정보

[찾아 바꾸기] 대화 상자

찾을 내용

찾을 내용을 입력합니다.

바꿀 내용

바꿀 내용을 입력합니다.

[바꾸기] 단추

[찾을 내용]을 찾아 블록으로 감싼 상태에서 [바꾸기] 단추를 누르면 해당 낱말이 [바꿀 내용]으로 바뀌면서, 다음 낱말을 찾아 블록으로 감싸 보여 줍니다.

[다음 찾기] 단추

[찾을 내용]을 찾아 블록으로 감싼 상태에서 [다음 찾기] 단추를 누르면 해당 낱말은 바꾸지 않고, 다음 낱말을 찾아 블록으로 감싸 보여 줍니다.

[모두 바꾸기] 단추

[모두 바꾸기] 단추를 누르면 사용자에게 확인하지 않고 지정된 범위 안에서 [찾을 내용]에 해당하는 낱말을 모두 찾아서 한꺼번에 [바꿀 내용]으로 바꾼 다음, 몇 번 바꾸기를 실행하였는지 알려 줍니다.

[닫기] 단추

찾기 실행을 종료하고 [찾기] 대화 상자를 닫습니다.

[찾기]나 [찾아 바꾸기]는 주로 여러 번 반복하여 실행하는 경우가 많기 때문에 사용자가 [닫기] 단추를 눌러 대화 상자를 닫을 때까지는 자동으로 닫히지 않고 화면에 계속 남아 있습니다.

참고

블록 설정하고 찾아 바꾸기

문서의 일부에서만 찾아 바꾸기를 하려면, 먼저 원하는 부분을 블록으로 설정한 다음 [찾아 바꾸기]를 실행합니다.

블록으로 설정된 부분에서 찾아 바꾸기를 끝낸 뒤에는 다음에서도 찾아 바꾸기를 계속할 것인지 사용자에게 확인합니다.

실행 범위

현재 커서의 위치에서 [찾을 방향]에 따라 [아래쪽]이면 문서의 끝까지, [위쪽]이면 문서의 처음까지 찾아 바꾸고, 나머지 부분에서도 계속 찾을지 사용자에게 확인합니다. [문서 전체]이면, 현재 커서의 위치부터 문서의 끝까지 찾은 다음 이어서 문서의 처음부터 현재 커서 위치까지 찾습니다.

[찾아 바꾸기] 대화 상자는 자동으로 닫히지 않음

[찾아 바꾸기]는 주로 여러 번 반복하여 실행하는 경우가 많기 때문에 사용자가 [닫기] 단추를 눌러 대화 상자를 닫을 때까지는 자동으로 닫히지 않고 화면에 계속 남아 있습니다.

활용

[바꾸기] 취소

[되돌리기]를 실행하면 [찾아 바꾸기]를 실행한 것을 원래대로 되돌릴 수 있습니다.
