[서식-글자 모양]의 [확장] 탭에서는 글자에 밑줄, 취소선, 테두리 등을 설정할 수 있습니다.

자세한 내용

[글자 모양] 대화 상자의 [확장] 탭

밑줄

밑줄의 [위치]와 [모양], [색] 등을 지정하여 글자에 다양한 밑줄을 그을 수 있습니다.

취소선

취소선의 [모양]과 [색]을 지정하여 글자에 다양한 취소선을 그을 수 있습니다.

테두리
  • 종류

    다양한 선 목록 중에서 원하는 글자 테두리의 선 종류를 선택합니다.

  • 굵기

    글자 테두리의 선 두께를 설정합니다.

  • 색

    색상표를 눌러 색상 팔레트가 나타나면 글자 테두리의 선 색을 선택합니다.

배경
  • 면 색

    색상표를 누르면 나타나는 색상 팔레트에서 글자의 배경색을 선택합니다.

  • 무늬 색

    색상표를 누르면 나타나는 색상 팔레트에서 글자의 배경을 채울 무늬 색을 선택합니다.

  • 무늬 모양

    [무늬 모양] 목록에서 글자의 배경을 채울 무늬 모양을 선택합니다.
