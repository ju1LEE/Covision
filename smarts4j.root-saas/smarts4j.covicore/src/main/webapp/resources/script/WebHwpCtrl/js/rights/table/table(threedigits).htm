1,000단위 구분 쉼표

표 안의 숫자에 일괄적으로 1,000단위마다 자릿점을 넣을 때 사용합니다.

[1,000단위 구분 쉼표-자릿점 넣기]를 하면 숫자를 감지하여 1,000단위마다 자릿점(,)을 넣어 주고 [1,000단위 구분 쉼표-자릿점 빼기]를 사용하면 숫자에 1,000단위로 사용된 자릿점(,)을 일괄 삭제해 줍니다.

따라 하기

  1. [표-표 만들기]를 실행합니다.
  2. [표 만들기] 대화 상자가 나타나면 줄 수와 칸 수를 지정하고 [만들기] 단추를 누릅니다.
  3. 만들어진 표에 1,000단위 숫자를 입력합니다.
  4. 표 안에 커서를 넣고 <F5>를 세 번 눌러 표 전체에 [셀 블록]을 설정합니다.
  5. [표-1,000단위 구분 쉼표-자릿점 넣기]를 누릅니다.
  6. 셀 블록으로 설정된 내용 중 숫자가 입력된 셀에 1,000단위마다 자릿점(,)이 입력됩니다.
  7. 다시 <F5>를 세 번 눌러 표 전체에 [셀 블록]을 설정합니다.
  8. [표-1,000단위 구분 쉼표-자릿점 빼기]를 누릅니다.
  9. 좀 전 입력되었던 자릿점(,)이 모두 삭제됩니다.
