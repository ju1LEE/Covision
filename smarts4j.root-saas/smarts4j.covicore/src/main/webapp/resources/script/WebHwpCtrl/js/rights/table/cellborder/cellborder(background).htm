셀 테두리/배경: 배경 <C>

[셀 테두리/배경] 대화 상자의 [배경] 탭에서는 셀의 배경을 색깔과 무늬 또는 그림으로 채우기 등을 지정하여 다양한 채우기 효과를 낼 수 있습니다.

[배경] 탭의 [적용 범위] 항목은 [표-셀 테두리/배경-각 셀마다 적용]을 실행한 경우에만 나타나는 항목이며, 현재 표의 셀 블록 설정 상태에 따라 [모든 셀, 선택된 셀]을 범위로 제시합니다.

따라 하기

  1. 색을 지정할 셀에 커서를 놓고 <F5>를 눌러 셀 블록을 설정한 후 [표-셀 테두리/배경-각 셀마다 적용]을 실행합니다.
  2. [셀 테두리/배경] 대화 상자가 나타나면 [배경] 탭을 선택합니다.
  3. [색]을 선택하고 [면 색] 항목에서 색상표를 누릅니다.
  4. 색상 팔레트가 펼쳐지면 원하는 배경색을 마우스로 선택하고 [설정] 단추를 누릅니다.
  5. 셀의 바탕이 선택한 색으로 채워집니다.

자세한 정보

[셀 테두리/배경] 대화 상자의 [배경] 탭

색 채우기 없음

[색]이나 [그림] 채우기 효과를 적용하지 않을 경우에는 [색 채우기 없음]을 선택합니다.

색
  • 면 색

  • 색상표를 눌러 색상 팔레트가 나타나면 배경 색을 선택합니다.

그림

그림 파일을 불러와 배경을 그림으로 채웁니다.

  • 그림 파일

  • [열기] 아이콘을 눌러 배경을 채울 그림 파일을 불러옵니다.

  • 채우기 유형

  • 배경을 그림으로 채울 때, 바둑판식으로 채울 것인지, 크기에 맞추어 채울 것인지, 쪽 가운데에 넣을 것인지 목록에서 채우기 유형을 선택합니다.

  • 그림 효과

  • 불러온 그림에 대해 회색조 처리를 하거나, 흑백 효과 등을 줄 수 있습니다.

  • 밝기

  • 선택한 그림 파일의 밝기를 조절합니다. 밝기의 단계는 -100부터 100까지 설정할 수 있습니다.

  • 대비

  • 선택한 그림 파일의 명암을 조절합니다. 명암의 단계는 -100부터 100까지 설정할 수 있습니다.

참고

셀 단위로 적용

셀 배경은 각 셀 단위로 적용됩니다. 셀 블록을 설정하고 셀 배경 그림을 넣더라도 하나의 그림이 모든 셀에 걸치는 배경으로 사용되는 것이 아니라, 각 셀마다 선택한 그림이 배경으로 나타납니다.

하나의 그림을 표의 모든 셀에 걸치는 배경으로 사용하고 싶을 때에는 [셀 테두리/배경-하나의 셀처럼 적용]을 실행하십시오.

셀 블록 상태에서 적용

현재 셀에만 테두리/배경을 적용할 때에는 셀 블록 상태에서 [셀 테두리/배경]을 실행시키십시오.

적용하려는 셀을 블록 설정하지 않고 커서만 위치시킨 후 [셀 테두리/배경]을 실행하면 전체 셀에 적용됩니다.
