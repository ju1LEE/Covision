한 수준 증가/감소 <Ctrl+Num -/+>

문단 번호는 번호는 최상위 수준인 1수준부터 최하위 수준인 10수준까지 다단계 번호를 부여할 수 있습니다.

[서식-한 수준 증가/감소] 기능을 이용하면 문단 번호의 수준을 손쉽게 바꿀 수 있습니다.

예를 들어, 1수준의 문단 번호를 한 수준 감소시키면 2수준이 되고, 10수준의 문단 번호를 한 수준 증가시키면 9수준이 됩니다.

따라 하기

문단 번호 한 수준 감소시키기 <Ctrl+Num +>

  1. 문단 번호를 적용할 문단을 블록으로 선택한 [서식-문단 번호 모양]을 눌러 문단 번호를 적용합니다.

  2. 둘째 문단에 커서를 놓고 [서식-한 수준 감소]를 누릅니다.

  3. 둘째 문단의 문단 번호 수준이 한 단계 내려가면서 셋째 문단부터 문단 번호가 새로 매겨집니다.

  4. 각 문단에서 [한 수준 감소] 기능을 이용하여 문단 번호 수준을 조정합니다.

문단 번호 한 수준 증가시키기 <Ctrl+Num ->

  1. 끝 문단에 커서를 놓고 [서식-한 수준 증가]를 누릅니다.

  2. 문단 번호 수준이 한 단계 올라갑니다.
