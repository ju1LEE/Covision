셀 모양 복사

표 안에서의 [모양 복사]는 커서 위치의 글자 모양이나 문단 모양, 스타일뿐만 아니라, 현재 셀의 셀 속성이나 선 종류, 셀 배경까지 함께 복사해 두었다가 다른 셀에 그대로 덮어 씁니다.

따라 하기

  1. 본문의 표에서 모양을 가진 글자 바로 뒤에 커서를 놓고 [편집-모양 복사]를 실행합니다.
  2. [모양 복사] 대화 상자가 나타나면 [본문 모양 복사-글자 모양]을 선택하고, [셀 모양 복사] 항목에서 [셀 속성], [셀 테두리], [셀 배경]을 선택한 다음 [복사] 단추를 누릅니다.
  3. 다른 표나 셀에서 모양을 바꿀 셀을 셀 블록으로 설정한 다음 [편집-모양 복사]를 다시 실행합니다.
  4. 셀 블록으로 설정한 셀의 셀 속성과 글자 모양이 복사해 두었던 모양으로 바뀝니다.

참고

모양 복사 적용

복사한 셀 모양은 셀 안의 내용을 블록으로 설정하거나 셀 블록을 설정한 다음 [편집-모양 복사]를 실행하여 기억된 모양을 덮어 씁니다.

표 안에서만 사용

셀 속성, 셀 테두리, 셀 배경과 같은 셀 모양 복사 항목은 표 안에서 [편집-모양 복사]를 실행한 경우에만 선택할 수 있습니다. 표 밖에서 모양 복사 덮어 쓰기를 실행하면 복사해 놓은 셀 모양은 무시됩니다.

주의

셀 속성만 복사할 수는 없음

표 안에서의 [모양 복사]는 "글자 모양"이나 "문단 모양"을 복사하면서 "셀 속성"을 함께 복사할 것인지 선택하는 것입니다. "셀 속성"은 복사하지 않고 "글자 모양, 문단 모양, 스타일"은 복사할 수 있지만, "글자 모양, 문단 모양, 스타일"은 복사하지 않고 "셀 속성"만 복사할 수는 없습니다.

모양 복사, 표/셀 속성, 표/셀 속성: 테두리, 표/셀 속성: 배경, 글자 모양, 문단 모양
