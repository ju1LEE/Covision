사용자 정의 용지

[용지 종류-사용자 정의]를 선택하면 기본적으로 제공되는 용지 종류가 아닌 사용자가 원하는 용지 크기를 직접 설정할 수 있습니다.

따라 하기

  1. [파일-편집 용지] 또는 [쪽-편집 용지]를 실행합니다.
  2. [편집 용지] 대화 상자가 나타나면 [용지 종류] 목록 단추를 누릅니다.
  3. 편집 용지 종류 중에서 [사용자 정의]를 선택합니다.
  4. "폭"과 "길이"에 각각 "200"을 입력하고, [설정] 단추를 누릅니다.

참고

사용자 정의

최소 20×20mm부터 1188×1188mm 사이에서 임의의 용지 크기를 지정하여, 한 쪽에 들어갈 본문 내용을 원하는 크기로 조절할 수 있습니다.

다른 시스템에서 사용

사용자가 직접 크기를 지정한 용지를 이용하여 작성한 문서를 다른 시스템에서 불러오면 [용지 종류-사용자 정의]로 나타납니다.
