하이퍼링크

하이퍼링크는 문서와 문서, 이메일 또는 웹 페이지를 연결하는 기능입니다. 새 브라우저 창을 열어서 특정 웹 페이지를 참조하도록 할 수 있습니다.

하이퍼링크는 단어 또는 문장 전체에 설정할 수 있습니다.

따라 하기

웹 주소로 하이퍼링크

  1. 하이퍼링크를 삽입할 텍스트를 블록으로 선택합니다.
  2. [입력-하이퍼링크]를 누릅니다.
  3. [하이퍼링크] 대화 상자의 [연결 대상] 영역에서 [웹 주소] 항목을 선택합니다.
  4. 웹 주소 입력란에 URL을 입력한 다음 [확인]을 누릅니다.

이메일 주소로 하이퍼링크

  1. 하이퍼링크를 삽입할 텍스트를 블록으로 선택합니다.
  2. [입력-하이퍼링크]를 누릅니다.
  3. [하이퍼링크] 대화 상자의 [연결 대상] 영역에서 [이메일 주소] 항목을 선택합니다.
  4. 이메일 주소 입력란에 연결할 이메일 주소를 입력한 다음 [확인]을 누릅니다.

자세한 정보

[하이퍼링크] 대화 상자

표시할 문자열

문서에서 하이퍼링크를 연결할 문자열을 입력합니다. 하이퍼링크가 연결되면 밑줄이 그어진 파란색 글자로 변경됩니다.

연결 대상

  • 웹 주소

    하이퍼링크를 연결할 웹 주소를 입력합니다.

  • 전자 우편 주소

    받을 사람의 전자 우편 주소를 입력합니다.

참고

하이퍼링크 이동

문자열에 지정된 하이퍼링크로 이동하려면 하이퍼링크 표시 위에서 마우스 왼쪽 단추를 한 번 누르거나 <Enter>를 누릅니다. 개체에 지정된 하이퍼링크로 이동하려면 개체 위에 마우스 포인터를 놓고 글자판의 <Ctrl>을 누른 채 마우스를 한 번 누릅니다.

하이퍼링크 고치기

본문에 삽입된 하이퍼링크 표시 안에 커서를 놓고 마우스 오른쪽 단추를 누른 다음 [하이퍼링크 고치기]를 선택합니다.

하이퍼링크 지우기

글자에 연결된 하이퍼링크를 지우려면 하이퍼링크 표시 안에 커서를 놓고 마우스 오른쪽 단추를 누른 다음 빠른 메뉴에서 [하이퍼링크 지우기]를 선택합니다.

도형, 그림과 같은 개체에 연결된 하이퍼링크를 지우려면 마우스 오른쪽 단추를 누른 다음 빠른 메뉴에서 [하이퍼링크 지우기]를 선택합니다.
