블록 계산식

표 안에서 셀 블록을 설정한 후 "블록 합계, 블록 평균, 블록 곱" 등의 계산 결과를 쉽게 구할 수 있습니다. [블록 계산식]은 성적표나 가계부와 같이, 값을 모두 입력한 후 총합이나 평균을 한 번에 계산하고자 할 때 이용하면 매우 편리합니다.

참고

하나의 셀에서 하나의 숫자만 계산함

셀 안에 여러 개의 숫자들이 입력되어 있더라도 그 셀에서 맨 처음 입력된 숫자만 블록 계산식에 반영합니다.

주의

제한

하나의 셀만 셀 블록으로 설정하거나 불연속적인 셀 블록을 설정한 경우에는 [블록 계산식]을 사용할 수 없습니다. 또한 블록 계산 결과를 자동으로 입력할 오른쪽이나 아래의 빈 셀이 없을 때에도 블록 계산 결과를 얻을 수 없습니다.

관련 기능

블록 합계, 블록 평균, 블록 곱, 표 만들기
