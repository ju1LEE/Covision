찾기/바꾸기

문서에서 특정 글자나 내용을 찾을 때 사용합니다. 영문은 대소문자를 구분하여 검색을 수행할 수 있습니다.

특정 내용 찾기

문서 내에서 특정 단어나 내용을 찾으려면 다음을 실행합니다.

  1. [편집-찾기-찾기]를 실행합니다.
  2. [찾기] 대화 상자가 나타나면 [찾을 내용]에 현재 문서의 내용 중에 찾고자 하는 문자열을 입력합니다.
  3. 마우스로 [다음 찾기] 단추를 누르거나, 글자판의 <Enter>를 누릅니다.
  4. [찾을 내용]에 해당하는 낱말을 찾아 블록으로 감싸 보여 줍니다. 만일 찾고자 하는 문자열이 없다면, 찾는 내용이 없다고 알려 줍니다.
  5. 마우스로 [다음 찾기] 단추를 누르거나 글자판의 <Enter>를 누르면 같은 내용을 다음에서 계속 찾습니다.

다른 내용으로 바꾸기

현재 편집하고 있는 문서에서 특정한 내용을 찾아 다른 내용으로 바꿉니다. 문서의 특정 내용을 찾아서 바꾸려면 다음을 실행합니다.

  1. [편집-찾기/바꾸기-찾아 바꾸기]를 누릅니다.
  2. [찾아 바꾸기] 대화 상자가 나타나면 [찾을 내용] 입력 상자에 찾을 내용을 입력합니다.
  3. [바꿀 내용] 입력 상자에 바꿀 내용을 입력한 다음 [바꾸기] 단추를 누릅니다. 찾은 내용을 한 번에 바꾸려면 [모두 바꾸기] 단추를 누릅니다.
