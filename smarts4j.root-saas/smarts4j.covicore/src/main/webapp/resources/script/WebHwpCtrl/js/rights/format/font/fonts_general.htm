글자 모양: 기본

[서식-글자 모양] 대화 상자의 [기본] 탭에서는 글자의 크기, 글꼴, 장평, 자간, 글자 속성, 글자 색 및 음영 색 등을 설정할 수 있습니다.

자세한 내용

[글자 모양] 대화 상자의 [기본] 탭

기준 크기

글자 크기를 키우거나 줄입니다. 글자 크기는 1pt부터 현재 편집 용지에 가득 차는 크기까지 제한이 없으며, 0.1포인트 단위로 지정할 수 있습니다.

언어별 설정
  • 언어

    [대표], [한글], [영문], [한자], [일어], [외국어], [기호], [사용자] 중에서 언어를 선택합니다.[대표] 글꼴은 언어별 글꼴에 대하여 서로 어울리는 글꼴끼리 짝지어 집합으로 만들어 놓은 것입니다.

  • 글꼴

    [언어]에서 선택한 언어 종류에 대하여 글꼴의 대표 이름으로 한꺼번에 지정하거나 각각의 언어별로 다른 글꼴을 지정할 수 있습니다.

  • 상대 크기

    기준 크기를 먼저 지정한 다음 언어별로 서로 다른 글자 크기를 지정합니다. 한글과 영문, 한자 등을 섞어 쓸 때 언어별로 적당한 상대 크기를 정해 놓고 사용하면 보다 글자의 크기와 모양을 고르게 표시할 수 있습니다.

  • 장평

    장평(글자의 가로/세로 비율)을 조정하여 글자의 크기는 그대로 유지하면서 글자의 가로 폭을 줄이거나 늘려서 글자 모양에 변화를 줄 수 있습니다.

  • 글자 위치

    글자의 기준선을 기준으로 글자를 위나 아래로 움직여 글자 위치를 조정합니다.

  • 자간

    자간(글자와 글자 사이의 간격)을 조절합니다. 각 글자 사이의 간격을 알맞게 조절하면 읽기 편한 문서를 만들 수 있습니다.

속성

[진하게], [기울임], [밑줄], [외곽선], [양각], [음각] 등의 아이콘을 눌러 다양한 글자 속성을 설정합니다.

  • 글자 색

    색상표를 누르면 나타나는 색상 팔레트에서 원하는 글자 색을 선택합니다.

  • 음영 색

    색상표를 누르면 나타나는 색상 팔레트에서 원하는 음영 색을 선택합니다.
