표/셀 속성: 셀

[표/셀 속성] 대화 상자의 [셀] 탭에서는 현재 셀이나 셀 블록으로 설정한 셀에 대하여 셀의 너비와 높이, 안 여백 지정, 세로 정렬 위치, 셀 보호, 제목 셀 설정 등의 셀 속성을 지정합니다.

따라 하기

  1. 셀 속성을 지정할 셀들을 셀 블록으로 설정하고 [표-표/셀 속성]을 실행합니다.
  2. [표/셀 속성] 대화 상자가 나타나면 [셀] 탭을 선택합니다.
  3. [안 여백 지정]을 선택하고, 왼쪽/오른쪽, 위쪽/아래쪽의 여백 값을 각각 입력합니다.
  4. [설정] 단추를 누르면 셀 테두리와 셀 안의 내용 사이의 간격이 설정한 값만큼 떨어집니다.

자세한 정보

[표/셀 속성] 대화 상자의 [셀] 탭

셀 크기 적용

현재 셀의 [너비]와 [높이]를 설정합니다. 셀 크기는 줄 칸 단위로 적용됩니다. 즉, [셀 크기 적용]을 선택해 [너비]와 [높이]를 각각 설정하더라도 현재 셀만 해당 너비와 높이를 갖는 것은 아닙니다. 현재 셀이 속한 줄 전체의 높이와 칸 전체 너비가 똑같은 너비와 높이를 갖습니다.

셀 크기를 지정할 때에는 현재 너비와 높이보다 작은 값을 설정할 수는 없습니다.

안 여백 지정

[안 여백 지정]을 선택해 셀의 안쪽 여백 값을 설정하면, [표 여백]에서 지정한 여백 값을 무시하고 셀 안쪽 여백 값에서 설정한 값만을 따릅니다.

속성
  • 셀 보호

  • 기본이 되는 문서 양식(form)을 만들어 놓고 항상 그 표의 일정 셀에 대하여 고정된 상태로 사용하기 위해서 [셀 보호] 기능을 선택합니다. 셀 보호가 설정된 셀에는 커서가 들어가지 않습니다.
    이를 해제하기 위해서는 셀 보호가 되어 있지 않은 셀에서 셀 블록 상태를 만든 다음 셀 보호가 된 셀로 이동합니다. 여기서 [표-표/셀 속성]을 실행하여 [셀 보호] 선택을 해제하십시오.

  • 제목 셀

  • 선택한 셀에 대하여 [제목 셀]을 선택해 놓으면 여러 쪽에 걸친 표를 만들 때 제목 줄이 자동으로 반복될 수 있도록 지정할 수 있습니다. [제목 셀]을 선택해야만 [표] 탭의 [제목 줄 자동 반복]이 동작합니다.
    단, [제목 셀]을 지정할 때에는 표의 첫 번째 줄을 반드시 포함해야 합니다. 첫 번째 줄을 제외하고 두 번째 줄을 [제목 셀]로 지정하면 [제목 줄 자동 반복]이 동작하지 않습니다.

  • 세로 정렬

  • 셀 안에 입력된 내용을 글상자 위쪽, 아래쪽, 가운데 중 어느 위치를 기준으로 정렬할지 지정합니다.

    아이콘 이름 설명
    위 셀 위쪽을 기준으로 정렬합니다.
    가운데 셀 가운데를 기준으로 정렬합니다.
    아래 셀 아래쪽을 기준으로 정렬합니다.
필드
  • 필드 이름

  • 외부 프로그램에서 한/글 문서 안의 자료를 읽고 쓸 수 있도록(COM 기능) 셀에 이름을 부여합니다.

  • 양식 모드에서 편집 가능

  • 셀을 삽입할 때 [양식 모드에서 편집 가능]을 선택하면 양식 모드에서도 삽입된 셀을 편집할 수 있도록 허용합니다. 양식 모드란 다른 사람이 문서 내용을 변경할 수 없도록 문서를 보호하며 양식 모드에서 편집 속성을 가진 개체만 편집이 가능하도록 허용하는 모드입니다. 이 기능은 외부 프로그램에서 한/글을 제어할 때 사용합니다.
