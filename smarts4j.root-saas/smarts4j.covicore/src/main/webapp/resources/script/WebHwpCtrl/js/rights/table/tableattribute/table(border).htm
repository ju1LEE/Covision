표/셀 속성: 테두리

[표/셀 속성] 대화 상자의 [테두리] 탭에서는 표 테두리의 종류, 굵기, 색 등을 설정할 수 있습니다.

자세한 정보

[표/셀 속성] 대화 상자의 [테두리] 탭

테두리
  • 종류

  • 11개의 선 목록 중에서 원하는 표 테두리의 선 종류를 선택합니다.

  • 굵기

  • 표 테두리의 선 두께를 설정합니다.

  • 색

  • 색상표를 눌러 색상 팔레트가 나타나면 표 테두리의 선 색을 선택합니다.

  • 셀 간격

  • 표의 각 셀과 셀 사이의 간격입니다. 값을 직접 입력하거나 증감 단추로 간격을 설정합니다.

적용 영역

테두리 종류, 굵기, 색을 설정하고 적용 영역에서 [모든 테두리], [왼쪽], [오른쪽], [위쪽] 또는 [아래쪽] 아이콘을 누르면 해당 위치에 선 종류가 적용됩니다.

참고

셀 간격

[표/셀 속성-테두리]에서 설정한 표 테두리는 [표/셀 속성-표]의 [셀 간격]에 값을 입력해 주어야만 나타납니다.
